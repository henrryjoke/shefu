# Changelog

## [0.2.1] — 2026-05-28

### Changed
- **起卦引导流程重写**：不再让用户选择起卦方式，不再让用户手摇铜钱（除非明确指定）
- **默认起卦方式改为大衍筮法**：新增 `scripts/qigua_dayan.py`，模拟蓍草四营十八变
- 起卦前引导改为委婉确认需求（"心里想的事已经清楚了吗？给几个关键词就好"）
- 用户任何回应都视为触发起卦信号，Agent 静默执行计算
- 六爻纳甲摇卦仅当用户明确指定"铜钱""硬币""摇卦"时使用
- 梅花易数仅当用户明确指定"梅花""梅易""数字""时间""物象"时使用

### Added
- 大衍筮法起卦引擎（`scripts/qigua_dayan.py`）：49根蓍草随机分堆，四营三变成一爻，十八变成一卦

## [0.2.0] — 2026-05-28

### Added
- 梅花易数起卦引擎（`scripts/qigua_meihua.py`）：时间起卦 + 数字起卦
- 六爻纳甲起卦引擎（`scripts/qigua_liuyao.py`）：铜钱摇卦 → 纳甲装爻 → 六亲 → 世应 → 六神
- 历法模块（`scripts/lifa.py`）：年月日干支、月建日建计算
- 64卦数据库（`data/guaci_db.json`）
- 纳甲数据表（`data/najia_table.json`）
- 起卦测试界面（`qigua_test.html`）
- SKILL.md：新增起卦入口、意识聚焦提示、默认六爻手摇优先级

### Fixed
- 体用关系判断反转
- 铜钱老阳/老阴规则（三背=老阳, 三字=老阴）
- 爻位显示顺序（上爻在上、初爻在下）
