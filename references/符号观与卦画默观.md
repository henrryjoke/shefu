## 一、旧符号观的根问题

当前 skill 的核心符号观可以概括为一句话：

> **符号 = 标签 → 查知识库 → 返回语义列表 → 组织输出**

这个模式有三个根问题：

| 问题 | 表现 | 札记中的批判 |
|---|---|---|
| **符号当物看** | 乾=天，坤=地，震=雷…把八卦当成 8 个名词 | 玄明札记#5："八卦不止天地雷风"——八卦是八种气的属性，不是八种物 |
| **含义靠检索** | 28 个 JSON + context 分类，推演 = `query("官鬼", "career")` | 玄明札记#4："象与理不是两截，是一体之表里"——查出来的不是理，是辞 |
| **卦义固定** | 符号有固定含义，语境只决定取哪个子集 | 玄明札记#4："包括不是罗列，是同构之容纳"——同一卦在不同语境下不是「不同侧面」，而是本质上就「成为」不同的东西 |

## 二、新符号观的总纲：爻位即气位

> 玄明札记#1：「卦画中的每一根爻，不是摆在那里的记号，而是气的位置、方向与势态的记录。爻位即气位——阳爻在哪里，阳气就在哪里；阴爻在哪里，阴气就在哪里。」

这是整个新符号观的基石。一句话足以推翻旧模式：

- 旧：这个符号「代表」什么？
- 新：这个位置的气「在做什么」？

## 三、浑仪之喻——卦画的本质

> 玄明札记#1引用陈抟：「凡此卦画，皆所以写天地万物之理于目前，亦若浑仪之器也。」

浑仪（浑天仪）不是天，但天象尽在其中。卦画不是天地，但天地之理尽在其中。

**这个比喻对 skill 的意义**：

| 浑仪 | 卦画 | Skill 中的对应 |
|---|---|---|
| 铜环 | 六爻 | 卦画本身 |
| 看浑仪需要训练 | 观卦脉需要功夫 | AI 推演就是「会看浑仪的人」 |
| 不会看的人只看到铜铁 | 不会观的人只看到横线 | 当前 Layer 5 查词典 = 在铜环上贴标签 |

## 四、八种德性：从结构中读出的功能态

### 4.1 德性 = 爻位结构的直接呈现

> 玄明札记#5：「每一个训释字，都直接对应一种爻位结构。性情不是外加的标签，而是爻位结构的直接呈现。」

| 卦 | 结构 | 德性（功能态） | 札记中的动态描述 |
|---|---|---|---|
| ☰ 乾 | 三阳无阴 | **健** | 纯阳自健之振，浑然不息、无偏无向 |
| ☷ 坤 | 三阴无阳 | **顺** | 纯阴自顺之容，寂然涵容、深长无间 |
| ☳ 震 | 一阳在二阴下 | **动** | 阳气在底部被压制，必然暴起冲破——**力的爆发** |
| ☴ 巽 | 一阴在二阳下 | **入** | 阴气从根基处悄然渗入，如地下水沿墙基——**质的浸染** |
| ☵ 坎 | 一阳在二阴中 | **陷** | 阳气被阴夹持，陷于中央进退两难——**力的受困** |
| ☲ 离 | 一阴在二阳中 | **丽** | 阴附于阳体核心，转化阳能为光——**质的转化** |
| ☶ 艮 | 一阳在二阴上 | **止** | 阳气走完全程到达顶端，力尽而止——**力的止息** |
| ☱ 兑 | 一阴在二阳上 | **说/悦** | 阴气盈满溢出散布，化为润泽——**质的溢散** |

### 4.2 阳卦与阴卦：两种根本动势

> 玄明札记#5：「阳气做事像锤——一锤一锤地敲，敲不动了就停。阴气做事像水——从缝隙中渗入，浸透每个角落，满溢后自然流散。」

| | 阳气入阴体（震坎艮） | 阴气入阳体（巽离兑） |
|---|---|---|
| 方式 | **冲**——以力破局 | **渗**——以柔穿刚 |
| 与主体的关系 | 对抗——始终在突围 | 融合——逐渐转化阳 |
| 全程 | 暴起→受困→止息（力盛→力衰） | 渗入→转化→溢散（质微→质满） |
| 观法 | 看力的轨迹——如箭飞行，一目了然 | 看质的弥漫——如墨散水，须耐心注视 |

### 4.3 乾坤 ≠ 六子：本体之脉 vs 流行之脉

> 玄明札记#3：「六子之脉是流行之脉（有方向、有节奏），乾坤之脉是本体之脉（浑然自运、无迹可循）。」

| | 六子之脉（流行） | 乾坤之脉（本体） |
|---|---|---|
| 性质 | 阴阳交错、有方向有节奏 | 纯气自运、浑然无间 |
| 可见性 | 如河流水纹，可循迹而观 | 如大海潮涌，浑然一体 |
| 来源 | 乾坤交错而生（破体） | 本之自然，无所假合（真体） |
| 动态 | 偏陂反侧——不平之气 | 圆融和粹——平气 |

**乾坤对待**——纯阳自健 + 纯阴自顺，两者之间形成宇宙间最根本的势差，这就是「一切流行之脉的母脉」。

## 五、观不是推——AI 解卦的认知论革命

> 玄明札记#1：「观卦脉不是推理，是直观。观者，直面而照见。你看到坎画，不需要经过一套推理程序才得出'月为地之气'——如果你真正学会了观脉，那一阳陷于二阴的象，会直接在你心中唤起'地中阳气凝而为月'的映照。」

这对 AI 意味着什么？AI 天然是一个「推理机」——它拿到输入后，经过 token-by-token 的概率推理生成输出。但《正易心法》要的不是「推导出什么」，而是「照见了什么」。

**实操方案**：把观脉和推演分开——

| 步骤 | 方式 | 札记依据 |
|---|---|---|
| **观脉** | 直面卦画结构，描述你看到了什么（不做翻译、不做解释、不联系事项） | "观者，直面而照见" |
| **落处** | 从结构中对应到天地/人身/人伦/治理的具体落处 | "须知落处，方有实用" |
| **推演** | 将落处的理「翻译」到用户事项语境 | 这就是当前 skill 做的事——但不是第一步，是最后一步 |

## 六、同构容纳——「包括万象」的真正机制

> 玄明札记#4：「包括不是罗列，是同构之容纳。卦画的本质是气脉结构的模型。天地万物之中，凡与此气脉结构同构者，皆可落入此卦。」

| 维度 | 坎 ☵ 的同构落处 |
|---|---|
| 自然 | 月为地之气（阳气凝于阴中） |
| 人身 | 肾为生命之根（元气藏于肾） |
| 人伦 | 中男承上启下（陷于枢纽） |
| 治理 | 行政居中执行（陷于日常而持守） |

**一脉三现（四现）**：同一条卦脉（阳陷于阴中），落于天则见月，落于身则见肾，落于家则见中子，落于国则见行政。不是四套系统，是一脉四现。

**这对 skill 的影响**：类象展开不应该只在一个语境维度展开。每个关键符号应至少展示 2-3 个维度的同构落处，让用户看到「同一个气脉结构在不同层面如何呈现」。

## 七、落处——全书的总方法

> 玄明札记#1 归纳的四步链条：
> ```
> 包括万象 ──→ 须知落处 ──→ 观于卦脉 ──→ 理则昭然
>   ①            ②            ③            ④
> ```

| 步骤 | 含义 | Skill 实现 |
|---|---|---|
| ① 包括万象 | 卦画是气脉的全息模型，同构者皆可落入 | 理论前提（不需显式步骤） |
| ② 须知落处 | 找到气在天地人身中的实落处 | **新增 Layer 4.5 卦画默观**：先看到气在哪里、在做什么 |
| ③ 观于卦脉 | 直观爻位中的气脉运行轨迹 | Layer 4.5 的延续：从爻位读出德性与动势 |
| ④ 理则昭然 | 理非推导而出，因脉迹照见而自明 | Layer 5 改造：先描述结构→再查类象验证 |

**落处是枢纽**——没有落处，①只是理论，③只是观赏，④只是愿望。有了落处，整个链条才活了。

## 九、方向红线——合于自然者为正

> 玄明札记#4：「从卦脉的自然之理出发，推出它在社会、人伦、治理中的对应——这是正途。反过来，如果你先有了一个社会观念，然后去卦画中找依据，那就是附会。合于自然者为正，附会人为者为谬。」

这对 skill 的操作约束：

```
正确方向：卦画结构 → 读出德性与卦脉 → 找到自然落处 → 平移到用户事项语境
错误方向：用户问题 → 猜测答案 → 在卦画中找依据 → 套上术语
```

**合于自然**是全书不可动摇的判准。推演中的每一个判断，都应该能从卦画结构中回溯验证。

## 十、改造后的 Skill 符号观（终版）

### 10.1 五条原则

| # | 原则 | 来源 |
|---|---|---|
| 1 | **爻位即气位**——阳爻在哪里，阳气就在哪里；阴爻在哪里，阴气就在哪里 | 札记#1 |
| 2 | **八卦是德性，不是名词**——乾是健、坤是顺、震是动……从卦画结构中直接读出，不靠词典 | 札记#5 |
| 3 | **观不是推**——先静观结构，后做语境翻译。观是直观照见，不是逻辑推导 | 札记#1, #4 |
| 4 | **同构容纳**——一个卦脉结构在多个维度有同构落处。类象展开应多维度呈现，而非单语境检索 | 札记#4, #6 |
| 5 | **合于自然者为正**——每个推演判断都应能从卦画结构回溯验证。方向必须是：结构→自然→语境，不可反向 | 札记#4, #7 |

### 10.2 新工作流

```
Layer 1 起卦（梅花） → Layer 4.5 卦画默观（新增） → Layer 2 六爻纳甲 → Layer 5 象展开（改造）
                                                              │
                                                    纳甲补充约束（后续 Phase）
                                                              │
                                                    奇门六壬进阶（后续 Phase）
```

**Layer 4.5 卦画默观（新增步骤）**：
```
4.5.1 爻位即气位 —— 六爻阴阳分布一览
4.5.2 德性识别 —— 上下卦各属什么德性？（健顺动入陷丽止说）
4.5.3 卦脉初判 —— 阳气的动势（进/陷/止）？阴气的动势（入/丽/悦）？
4.5.4 本卦是哪种脉 —— 真体（乾坤）？流行（六子）？假合（重卦）？
4.5.5 态势初判 —— 这个卦的整体态势方向
4.5.6 落处初探 —— 气脉结构在自然/人身/人伦/治理维度的初步同构落处
```

**Layer 5 象展开（改造）**：
```
5.1 从默观结果出发，先描述「这个卦脉结构在做什么」
5.2 将结构动态映射到用户事项语境（"这件事的气脉态势类似…"）
5.3 多维度同构展示（至少 2-3 个维度）
5.4 查类象知识库 → 核对一致性
5.5 不一致 → 标注张力，不掩盖
```

### 10.3 类象知识库的重新定位

- **主数据源**（旧）→ **验证层**（新）
- JSON 结构暂时不动（降低回归风险），但调用定位改变
- 远期重构为「功能态→派生类象树」结构
