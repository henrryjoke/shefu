# 卦脉解读《说卦传》第十一章

> 方法：卦脉 = 爻位结构 → 德性（功能态）→ 同构落处（类象）
> 乾☰三阳纯奇=健 | 坤☷三阴纯偶=顺 | 震☳一阳在下=动 | 巽☴一阴在下=入
> 坎☵一阳在中=陷 | 离☲一阴在中=丽 | 艮☶一阳在上=止 | 兑☱一阴在上=说

---

## 一、按八卦逐条卦脉解读

### 1. 乾 ☰ —— 三阳纯奇 → 健

> **结构**：三爻全阳，无阴掺杂。阳气纯一无间。
> **德性**：**健**——纯阳自健之振。不依赖任何外力，自我驱动，永恒运行。
> **卦脉**：如鼓面之共鸣，处处齐动，无处为始，无处为终。

| 条目 | 落处域 | 卦脉推导 |
|---|---|---|
| 天 | 自然·天体 | 阳之极纯者为天。天行健，纯阳自运不息——这正是乾卦「健」在自然的第一落处 |
| 圜 | 自然·形状 | 纯阳之气运行无始无终、无往无来——其轨迹为圆。圜者，健行的几何形态 |
| 君 | 人伦·治位 | 君为施气之源。如乾之纯阳为六子之源，君主是万务之源头——「不动之一」 |
| 父 | 人伦·家庭 | 父为家之施气者。乾之健对应一家之主——纯阳施化，不偏不倚 |
| 玉 | 物用·珍宝 | 玉质纯而坚，光芒内含——如同纯阳之气凝为最纯之质。健之凝态 |
| 金 | 物用·金属 | 金为至坚之质，百炼不折——纯阳之健在物质层面极化为不折之刚 |
| 寒 | 自然·气候 | 纯阳至极反生寒——阳到极点，外热内凝。如高空寒冽，阳气纯而不散则寒 |
| 冰 | 自然·物态 | 寒之极凝为冰——纯阳之气凝为固态。冰坚而明，正是健之极凝 |
| 大赤 | 自然·颜色 | 纯阳之色为赤。赤者，阳之极——如正午之日色，纯阳无杂的视觉呈现 |
| 良马 | 动物·马 | 马健行千里——动物中最近乾之健者。良马纯健，力足而持久 |
| 瘠马 | 动物·马 | 瘠马瘦而骨健——健不一定满，纯阳可以极精瘦。瘦骨嶙峋中见纯健 |
| 驳马 | 动物·马 | 驳马色杂而力猛——健可以容杂色而不损其力。纯阳之体可纳异而仍健 |
| 木果 | 植物·果实 | 木果圆而实，内含生机——纯阳之气凝为一颗种子。果实是健的终点也是新健的起点 |

### 2. 坤 ☷ —— 三阴纯偶 → 顺

> **结构**：三爻全阴，无阳掺杂。阴气纯一无间。
> **德性**：**顺**——纯阴自顺之容。承载一切，为他者提供空间。
> **卦脉**：如深渊之蓄水，看似静止，实则暗流遍体，无所不至。

| 条目 | 落处域 | 卦脉推导 |
|---|---|---|
| 地 | 自然·大地 | 阴之极纯者为地。地势坤，纯阴自顺而承载万物——这正是「顺」在自然的第一落处 |
| 母 | 人伦·家庭 | 母为家之受化承载者。如坤之纯阴为万物孕育之场域——顺而厚载 |
| 布 | 物用·织物 | 布平坦铺展，覆盖包裹——纯阴之顺展开为平面。柔而广覆 |
| 釜 | 物用·容器 | 釜中空受物——纯阴以虚受实。釜之空，正是坤之容 |
| 吝啬 | 性状 | 阴主收藏，纯阴则极藏。吝啬者，只藏不散——顺之极藏态 |
| 均 | 性状 | 地无偏载，万物平等——纯阴之顺不拣择对象。均匀是顺的遍在表现 |
| 子母牛 | 动物·牛 | 母牛怀子、顺而多产——动物中最显顺之厚载。生生不息，顺育无穷 |
| 大舆 | 物用·车 | 大车承载万物而行——纯阴之顺在器物上的体现。舆者，承载而行 |
| 文 | 性状 | 文者，纹理、章采——纯阴展开为万物的可见纹路。顺之力把无形化为有形 |
| 众 | 性状 | 纯阴能容万象——众者，万物归焉。顺之所至，万象咸集 |
| 柄 | 物用·工具 | 柄为执持之本——纯阴之顺在手握处化为可执之体。万物有所执乃能行 |
| 黑 | 自然·颜色 | 纯阴之色为黑。黑者，阴之极——如深夜之色，纯阴无光的视觉呈现 |

### 3. 震 ☳ —— 一阳在二阴下 → 动

> **结构**：一阳在最底部，二阴在上一中。阳气被压在阴的最深处。
> **德性**：**动**——暴起冲破。如弹簧被压到极限，必然反弹。
> **卦脉**：阳气在阴的最深处积蓄力量，一朝爆发，天地为之一震。

| 条目 | 落处域 | 卦脉推导 |
|---|---|---|
| 雷 | 自然·气象 | 雷声骤至，天地一震——阳气在阴中暴起的自然落处。动之第一象 |
| 龙 | 动物·神兽 | 龙潜于渊、腾于天——阳气从最深处跃出。龙的飞腾即震之暴起 |
| 玄黄 | 自然·颜色 | 玄为天色、黄为地色——天地之气交激时的混沌之色。阳从阴中初出，色未纯 |
| 敷 | 自然·展开 | 阳气暴起后向外敷布——动的后续展开。如雷后春雨，遍洒大地 |
| 大涂 | 自然·道路 | 涂者，途也。大涂——阳气暴起开辟的通路。动之迹留下大道 |
| 长子 | 人伦·家庭 | 第一缕阳气入坤体而生长男。长子首当其冲——暴起开路者 |
| 决躁 | 性状 | 阳气积压后的暴起必然迅猛——决者果决，躁者急切。动之性情 |
| 苍筤竹 | 植物·竹 | 竹初生为笋，破土而出——阳气从地底暴起的植物版。其色苍（青） |
| 萑苇 | 植物·芦苇 | 芦苇丛生，风吹齐动——阳气催发的集体动势。细而多，生发之满 |
| 善鸣（马） | 动物·马 | 马之善嘶鸣者——阳气欲出之声响。善鸣即动的声学形态 |
| 馵足（马） | 动物·马 | 马之后左足白——阳气在底部的标记。动之初足，力从足起 |
| 的颡（马） | 动物·马 | 马之额有白章——阳气欲上冲的头部标记。阳在底而光已现于上 |
| 反生（稼） | 作物 | 种子先向下生根、再向上出苗——阳气在底部先积蓄再突破。动之两段 |
| 其究为健 | 性状·归趋 | 阳气从底部一路暴起到顶，最终化为纯阳的健——震之终极是乾 |
| 蕃鲜 | 性状 | 阳气暴起后的繁盛——动的成果：万物蕃盛而鲜明 |

### 4. 巽 ☴ —— 一阴在二阳下 → 入

> **结构**：一阴在最底部，二阳在上一中。阴气从阳的根基处潜入。
> **德性**：**入**——悄然渗入。如地下水沿墙基渗入建筑——不争不抗，顺势而行。
> **卦脉**：阴气趁阳气底部之虚，从最细微处渗入，如丝如缕，无孔不入。

| 条目 | 落处域 | 卦脉推导 |
|---|---|---|
| 木 | 自然·植物 | 木之根深入地下、枝叶散于空中——阴气从下渗入阳体的自然版。根即巽之入 |
| 风 | 自然·气象 | 风入林间，不是一吹而过，而是从树根到枝叶逐层穿透——阴气穿透阳体的全域 |
| 长女 | 人伦·家庭 | 第一缕阴气入乾体而生。长女率先入局——潜入根底者 |
| 绳直 | 物用·工具 | 绳以柔克刚，以细束粗——阴气潜入后用以矫正阳的偏斜。入而后有绳 |
| 工 | 人伦·职业 | 工匠以技巧入于物——以阴柔之巧深入阳刚之材。入的精工形态 |
| 白 | 自然·颜色 | 白者无色——阴气潜入到极致则褪去一切颜色。入之极则无色 |
| 长 | 性状 | 阴气潜入后持续蔓延——入的延伸效果。如藤蔓之长 |
| 高 | 性状 | 阴气从底部渗入后上行至顶——入的垂向延伸。如烟之升 |
| 进退 | 性状 | 阴气潜入时走走停停——入的不确定节奏。时进时退，时深时浅 |
| 不果 | 性状 | 潜入者不暴露意图，不轻易决断——入的迟疑性。阴柔故不果 |
| 臭 | 性状 | 气味无形，弥漫渗透——阴气入于阳体的嗅觉表现。无孔不入 |
| 寡发 | 人体·头部 | 阴气潜入消耗阳气→头发少。阴气从根基抽走阳的性能量 |
| 广颡 | 人体·面部 | 阴气入于底部→额头宽阔。阴气把空间撑开 |
| 多白眼 | 人体·眼部 | 阴气潜入的目光表现——眼多白，是阴气占据视觉的标记 |
| 近利市三倍 | 经济 | 阴气潜入商业运作——以柔穿刚，以无形渗入获利。入的商业形态 |
| 其究为躁卦 | 性状·归趋 | 巽之终极，潜入太深后阳气反弹→化为震之躁动。入极反动的辩证 |

### 5. 坎 ☵ —— 一阳在二阴中 → 陷

> **结构**：一阳在中间，二阴在上下。阳气被阴气夹持于中央。
> **德性**：**陷**——被困于核心。阳气深陷阴体内部，进退两难。
> **卦脉**：阳气进入阴体深处后发现上下左右皆是阴——突围需要力量，力量在此被消耗分散。虽陷，却是核心。

| 条目 | 落处域 | 卦脉推导 |
|---|---|---|
| 水 | 自然·基本 | 水流于低处，陷于沟壑——阳气陷于阴中的自然版。陷之第一象 |
| 沟渎 | 自然·水道 | 沟渎是水流之陷处——水汇聚于最低处。陷的地形形态 |
| 隐伏 | 性状 | 阳气陷于阴中不可见——陷的视觉特征。不显于外 |
| 矫輮 | 物用·工具 | 矫輮使曲者直——阳气虽陷于阴中，仍有矫正之力。陷中有健 |
| 弓轮 | 物用·器物 | 弓之弯曲、轮之圆转——陷中之阳的弯曲形态。阳屈于阴中而成用 |
| 加忧 | 人体·情绪 | 阳气陷于阴中→心思沉重。陷的心理对应——深深的忧虑 |
| 心病 | 人体·疾病 | 阳陷于心→心中郁结。心为阳脏，坎中一阳陷则心病 |
| 耳痛 | 人体·疾病 | 坎为耳，阳陷于耳→耳痛。陷在感官上的表现 |
| 血卦 | 人体·体液 | 血为红色液体（阴中含阳）——血在脉中流动如阳陷于阴中流动 |
| 赤 | 自然·颜色 | 坎中一阳之色——虽陷于阴中，阳仍发为赤色。陷中之光 |
| 美脊（马） | 动物·马 | 马之美脊——中脊为阳（坎中一阳的位置），陷于肉（阴）中 |
| 亟心（马） | 动物·马 | 马心急而力——坎中阳急于出困。陷中的焦躁 |
| 下首（马） | 动物·马 | 马首下垂——阳气陷而不扬。陷的体态 |
| 薄蹄（马） | 动物·马 | 马蹄薄——阳气陷于中，末端（蹄）薄。陷中力量不足达于末 |
| 曳（马） | 动物·马 | 马行拖曳——阳陷于阴中，行动缓慢沉重 |
| 多眚（舆） | 物用·车 | 车多故障——阳陷于阴体，器物易损。陷的器物状态 |
| 通 | 性状 | 坎中一阳贯穿阴阳——陷中有通。如水能穿石 |
| 月 | 自然·天体 | 月为地气之凝——阳陷于阴中凝为月华。坎之天体落处 |
| 盗 | 人伦·角色 | 盗者隐伏暗处——阳气陷于阴中潜藏。陷的社会角色 |
| 坚多心（木） | 植物·木 | 木之纹理坚密多结——阳气陷于木中凝为坚结。陷的木象 |

### 6. 离 ☲ —— 一阴在二阳中 → 丽

> **结构**：一阴在中间，二阳在上下。阴气附于阳体核心。
> **德性**：**丽**（附丽）——阴依附于阳体、因阳而明。如烛芯被蜡油点燃，自己不发光，火光却从阴处升起。
> **卦脉**：阴本身不发光，但居于二阳之间，阳的能量从上下汇聚于此阴——阴将不可见的阳能转化为可见的光明。

| 条目 | 落处域 | 卦脉推导 |
|---|---|---|
| 火 | 自然·基本 | 火发光发热——阳气在外、阴质在中。阴质将阳能转化为可见的光。丽之第一象 |
| 日 | 自然·天体 | 日体纯阳，但中有阴质——离之中阴将纯阳之热转化为能照万物的光。丽之天体落处 |
| 电 | 自然·气象 | 电光一闪——阴阳交激的瞬间。丽之瞬时爆发态 |
| 中女 | 人伦·家庭 | 第二缕阴气入乾体而生。中女居中而丽——居核心而附丽于家 |
| 甲胄 | 军事·防具 | 甲胄附于人身而护体——阴依附于阳而发挥保护功能。丽的防护形态 |
| 戈兵 | 军事·武器 | 戈兵锋利而明亮——阳在外，阴在内。丽的攻击形态 |
| 大腹 | 人体·腹部 | 腹部中空能容——阴在阳体中的容纳空间。丽的空间形态 |
| 乾卦 | 性状 | 离中阴质→干燥。阳在外炙烤，阴在内被烤干。丽之燥态 |
| 鳖 | 动物·水族 | 鳖外有坚壳（阳在外），内藏柔体（阴在内）——丽的甲壳动物版 |
| 蟹 | 动物·水族 | 蟹壳硬而内柔，横行——丽的外部刚硬、内部柔软。外壳即外阳 |
| 蠃 | 动物·水族 | 螺类壳旋而中空——阳在外螺旋包裹，阴在内虚空。丽的螺旋版 |
| 蚌 | 动物·水族 | 蚌壳双合，内含珍珠——阴在外阳之间孕育珍珠。丽之孕育态 |
| 龟 | 动物·水族 | 龟壳圆硬（外阳），体内柔（内阴）——丽的灵龟版 |
| 科上槁（木） | 植物·木 | 木之上部枯槁——阳在外太盛，阴在内被耗尽。丽之枯竭态 |

### 7. 艮 ☶ —— 一阳在二阴上 → 止

> **结构**：一阳在最顶部，二阴在下。阳气走完全程，到达阴体顶端。
> **德性**：**止**——力尽而止定。如登山者攀至峰顶——上面没有更高的地方了，止步。
> **卦脉**：阳气从底部一路突破（震）→陷于中段（坎）→最终到达顶端（艮）。不是失败，而是自然止定。

| 条目 | 落处域 | 卦脉推导 |
|---|---|---|
| 山 | 自然·地形 | 山为阳气自天而降、凝止于地表而成——止的自然第一落处 |
| 径路 | 自然·道路 | 小路止于山中——止的通行形态。径者，小止于山之途 |
| 小石 | 自然·矿物 | 小石为山体碎而止定者——止的微缩形态。石小而坚，止而不移 |
| 门阙 | 建筑·门户 | 门为内外之界——出入至此而止。阙者，止于门前的高台 |
| 果蓏 | 植物·果实 | 果实为植物生长之终点——止于果。草木之生，终于果蓏 |
| 阍寺 | 人伦·职守 | 守门者止人出入——止的社会角色。阍寺执守于门 |
| 指 | 人体·手部 | 手指为手之末端——阳气运行到肢体末端而止。止于指尖 |
| 狗 | 动物·家畜 | 狗守门止盗——止的动物形态。护门而止外人 |
| 鼠 | 动物·小兽 | 鼠藏于穴、止于暗处——止的隐蔽形态。止于暗穴不出 |
| 黔喙之属 | 动物·猛禽 | 黑喙猛禽——其喙为收敛（止）与攻击的结合。止的猎杀形态 |
| 坚多节（木） | 植物·木 | 木多节处为阳气止定之处——止的木象。节者，气聚而止 |

### 8. 兑 ☱ —— 一阴在二阳上 → 说/悦

> **结构**：一阴在最顶部，二阳在下。阴气走完全程，到达阳体顶端。
> **德性**：**说**（悦）——盈满而溢出散布。阴气无处可上，如水满自然漫出，化为润泽。
> **卦脉**：阴气从底部潜入（巽）→附丽转化（离）→最终到达顶端溢出（兑）。不是被迫溢出，而是盈满后的自然漫出。

| 条目 | 落处域 | 卦脉推导 |
|---|---|---|
| 泽 | 自然·水体 | 泽为水聚于低处→盈满后向四方漫出——阴气在阳体顶端溢出散布。说之第一落处 |
| 少女 | 人伦·家庭 | 第三缕阴气入乾体而生。少女最后呈现——以喜悦的姿态外显 |
| 巫 | 人伦·角色 | 巫以口通神——悦的表达。阴气溢散为言语、歌舞以通神灵 |
| 口舌 | 人体·器官 | 口为言语所出——阴气从顶端溢出为声。舌为口之辅 |
| 毁折 | 性状 | 阴气溢出后阳体被消耗→毁折。悦的另一面——散则损 |
| 附决 | 性状 | 阴气浮于阳体顶端→一触即脱。附着于上而随时可脱离 |
| 刚卤（地） | 自然·土地 | 地之盐碱——泽水蒸发后留下的刚硬咸土。悦之散尽后的剩余 |

## 三、卦脉解读《说卦传》第 11 章的总结

### 3.1 全章覆盖度

《说卦传》第 11 章共约 120+ 条类象。**每一条都可以从「结构→德性→功能态→落处」的卦脉链条中推导出来。**

不靠「查表」，不靠「口诀」，只看卦画中气在哪里、在做什么、这个动作在某个领域会表现为什么东西——这就是卦脉方法。

### 3.2 卦脉推导的通式

```
卦画结构 → 德性（1 字）→ 功能态描述（1 句）→ 落处翻译
                                                      ├── 自然落处：气在这个领域表现为 X
                                                      ├── 人伦落处：气在这个领域表现为 Y
                                                      ├── 动物落处：气在这个领域表现为 Z
                                                      └── ……
```

### 3.3 对比当前类象库

当前 bagua/ 下 8 个 JSON 文件的 contexts 分类（career/health/relationship/finance/general）是面向现代用户场景的，与《说卦传》的落处分类不同但互补：

| 《说卦传》落处 | 当前 JSON contexts |
|---|---|
| 自然·天体/气象/地形 | general.space, general.object |
| 人伦·家庭/治位/职业 | general.person, career.person |
| 动物 | 缺失（可加入 general.object） |
| 植物 | 缺失（可加入 general.object） |
| 人体 | health |
| 器物/军事/建筑 | general.object |
| 性状 | general.state |
| 颜色 | general.object |

**建议**：远期在 bagua JSON 中补充「自然落处」维度（天体、气象、动物、植物、人体、颜色），使类象库与《说卦传》的卦脉推演体系完全对齐。

### 3.4 卦脉方法对当下 Skill 的核心价值

1. **解释力**：不是「乾代表玉」，而是「乾之纯阳健气凝为至坚至纯之质 → 玉」
2. **泛化力**：遇到不在类象库中的新场景，可用卦脉推导新类象，而非束手无策
3. **一致性**：所有类象追溯到同一个结构根，不会出现自相矛盾的解释
4. **合于自然**：每一条都可以回答「为什么」，因为它是从气脉自然运行中照见的，不是人编的
