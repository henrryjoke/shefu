# 辨识问题库 · 射覆约束求解用

> **用途：** 当 L3 约束求解给出 🟡 中置信候选集时，AI 从此库中选择辨识问题。
> **原则：** 只问卦象给不出的维度；每次只问一个维度；二选一/对比式；最多两轮。

---

## 使用规则

```
给定候选集（3-5个）→ 识别它们共属的品类 → 从该品类选问题模板
→ 选最能区分剩余候选的那个问题维度
→ 问一问 → 用户回答 → 排除 → 决定是否需要第二问
```

## 品类一：点火/光源（离+震/离为主）

| | |
|---|---|
| **典型候选** | 打火机、火柴、蜡烛、LED灯、手电筒、点火器 |
| **最有效辨识维度** | 大小 + 能量来源 |

### 问题模板

**Q1（首选 — 能量来源）：**
> 「是用火（燃料/气体）的还是用电的？」

| 回答 | 排除 | 保留 |
|---|---|---|
| 用火 | LED灯、手电筒 | 打火机、火柴、蜡烛、点火器 |
| 用电 | 打火机、火柴、蜡烛、点火器 | LED灯、手电筒 |

**Q2（大小 — 若 Q1 未唯一）：**
> 「是手持小件还是需要放在桌面/墙上的？」

| 回答 | 排除 | 保留 |
|---|---|---|
| 手持 | 台灯、蜡烛台 | 打火机、火柴、手电筒 |
| 非手持 | 打火机、火柴 | 蜡烛、LED台灯 |

## 品类二：电子/电力（坎+震/坎为主）

| | |
|---|---|
| **典型候选** | 充电器、充电线、电池、充电宝、电源适配器、U盘、数据线 |
| **最有效辨识维度** | 形态 + 附属件 |

### 问题模板

**Q1（首选 — 形态）：**
> 「是块状的（充电头/充电宝）还是线状的（数据线）？」

| 回答 | 排除 | 保留 |
|---|---|---|
| 块状 | 充电线、数据线 | 充电头、充电宝、电池、U盘 |
| 线状 | 充电头、充电宝、电池、U盘 | 充电线、数据线 |

**Q2（若 Q1=块状，附属件）：**
> 「它插电的时候需要一根线连着吗，还是直接插在插座上？」

| 回答 | 排除 | 保留 |
|---|---|---|
| 直接插 | 充电宝、U盘 | 充电头 |
| 需要线 | 充电头 | 充电宝、U盘 |

**Q2（若 Q1=线状，结构）：**
> 「是两头都有插头的，还是一头固定不可拔的？」

| 回答 | 排除 | 保留 |
|---|---|---|
| 双头可拔 | 耳机线、固定接线 | 数据线（双头线） |
| 一头固定 | 双头数据线 | 带固定头的充电线 |

**Q1（首选 — 佩戴部位）：**

**Q2（若 Q1=头上，具体部位）：**
> 「是进入耳朵/遮住眼睛的，还是覆盖口鼻的？」

| 回答 | 排除 | 保留 |
|---|---|---|
| 入耳/遮眼 | 口罩 | 耳塞、耳机、眼镜 |
| 口鼻 | 耳塞、耳机、眼镜 | 口罩 |

**Q2（若 Q2 未排除到唯一，材质/重量）：**
> 「是硬的还是软的？」

| 回答 | 排除 | 保留 |
|---|---|---|

## 紧急回退策略

当候选品类不明确（跨越多个品类），或以上问题均不适合时：

**通用问题（按优先级）：**

```
1. 大小：「手握大小，还是更大/更小？」     ← 永远有效
2. 重量：「轻飘飘的还是有分量感的？」      ← 辨别密度
3. 材质触感：「硬的还是软的？」           ← 快速排除大类别
4. 用途：「工具/饰品/食物/其他？」         ← 类别确认
```

**只用通用问题的场景：** 候选来自 3 个以上不同品类，品类级问题无法一次性排除足够多。
