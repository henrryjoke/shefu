# 补充层：月建与卦气旺衰

> 定位：月建作用于八卦卦气，作旺相休囚死判断，调节卦脉解读中气的强弱。
> 边界：日建仅用于纳甲卦爻层，不适用于八卦卦气判断。

---

## 一、定位与边界

```
B1→B5 基础解卦方法        → 卦气的「质」（是什么气、怎么流、往哪落）
月建辅助                    → 卦气的「量」（当下这个月，气有多强？）
纳甲层（后续）             → 卦气的「位」（世应在哪里，六亲六神附加约束）
```

**月建不改变卦脉的解读方向，但改变卦脉的强弱权重。**

> 玄明例：「水泽节卦，坎水在上流动，兑在下满溢。申酉月兑旺坎相→雨水势大绵绵不绝。亥子月坎旺兑休→雨雪必不长久。」

## 三、月建如何融入卦脉解读

### 3.1 加权规则

月建的信息在 B1→B4 完成后叠加：

```
B1→B4 产出：上卦德性 + 下卦德性 + 落处翻译
                │
月建加权：查上卦五行旺衰 → 调节「上卦（外环境）气的强弱」
          查下卦五行旺衰 → 调节「下卦（内根基）气的强弱」
                │
加权后：上卦[旺/相/休/囚/死] + 下卦[旺/相/休/囚/死]
        → 重新评估体用力量对比
        → 调节落处翻译中的确定性/力度描述
```

### 3.2 加权对落处翻译的影响

| 月建状态 | 落处翻译调节 |
|---|---|
| 旺 | "气势正盛""正值其时""充沛有力" |
| 相 | "得时令生扶""有后续支撑""渐入佳境" |
| 休 | "气有所泄""后劲渐弱""需借外力" |
| 囚 | "为时令所制""伸展受限""强行则损" |
| 死 | "当令所克""极弱难显""静守为安" |

### 3.3 月建加权后的体用评估

B1 中体用关系是德性层面的（健顺动入陷丽止说）。月建加权后还需要做力量对比：

```
体卦月建状态 vs 用卦月建状态：
  · 体旺用弱 → 内在驱动力强于外部阻力，主动态势
  · 体弱用旺 → 外部环境强于内在驱动，被动态势
  · 体用均旺 → 内外皆强，事态活跃但可能冲突
  · 体用均弱 → 内外皆弱，事态沉寂需待时
```

## 四、示例：水泽节卦的月建推演

### 4.1 卦脉基础（B1→B4）

```
本卦：水泽节 ䷻
上卦坎 ☵ = 一阳陷于二阴中 → 德性「陷」——水气在上，流动而陷
下卦兑 ☱ = 一阴在二阳上 → 德性「悦/散」——泽气在下，满溢而散

卦脉解读：水在泽上，流动不居。坎水陷而流，兑泽满而溢。
整体态势：水气横行——内外皆水势，但有节（坎陷=受节，兑悦=乐节）。
```

### 4.2 不同月建的加权

#### 场景 A：申酉月（金旺）

| 卦 | 五行 | 月建申酉金 | 状态 |
|---|---|---|---|
| 上卦坎 | 水 | 金生水 | **相** ★★★★ |
| 下卦兑 | 金 | 同金 | **旺** ★★★★★ |

**卦脉加权解读**：
- 下卦兑金得月建旺气 → 泽水之源充沛（内在水气强劲）
- 上卦坎水得月建相气 → 流水有源不断（外在表现持续）
- **兑旺坎相 = 源头旺、流动强** → 水气源源不绝
- 落处翻译：雨水势大，持续时间长。因为有源头（兑旺）不断供给。

#### 场景 B：亥子月（水旺）

| 卦 | 五行 | 月建亥子水 | 状态 |
|---|---|---|---|
| 上卦坎 | 水 | 同水 | **旺** ★★★★★ |
| 下卦兑 | 金 | 金生水 | **休** ★★ |

**卦脉加权解读**：
- 上卦坎水得月建旺气 → 当前水势正盛（外在表现强烈）
- 下卦兑金被月建泄气 → 泽水之源已耗（内在供给不足）
- **坎旺兑休 = 当前猛、源头虚** → 来得快去得也快
- 落处翻译：雨水或雪势虽大但不持久。因为当前气足（坎旺）而后续无力（兑休），如同水库已近干涸而最后放一波水。

### 4.3 对比

| 维度 | 申酉月 | 亥子月 |
|---|---|---|
| 上卦坎 | 相（金生水，有源） | 旺（同水，气势盛） |
| 下卦兑 | 旺（同金，源充沛） | 休（金生水，源被泄） |
| 整体 | **持续强降雨** | **短时大雨/雪，迅速收止** |
| 卦脉不变 | 水气横行 | 水气横行 |
| 月建改变 | → 持久力 | → 爆发力 |

> 关键：**月建不改卦脉之质，只改卦气之量。**

## 六、边界重申

| 月建 | 日建 |
|---|---|
| 用于八卦卦气旺衰 | **不用于八卦** |
| 判断上下卦的当下强弱 | 仅用于纳甲卦爻（世应/六亲/六神层次） |
| 调节 B1→B4 落处翻译的力度 | 调节纳甲层的爻位旺衰 |

**两者不应混淆。** 日建对卦的影响通过纳甲六爻来体现，不直接作用于八卦的类象解读。
