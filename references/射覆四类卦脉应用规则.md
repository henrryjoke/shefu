# 射覆四类卦脉应用规则

> 基于17案例实证，总结本卦/互卦/变卦/动爻在射覆中各自的责任边界
> 原理：同一组六爻的气态被四个取爻规则切成四个窗口，各自看到不同的切面
> v0.7.1-dev 新增：月建对互卦上下经卦的调节机制 + 坎囚存象梗脊 + 震休纸壳茶叶

---

## 速查总表

| 窗口 | 取爻 | 同构何象 | 表达什么 | 对应维度 | 确定性 |
|------|------|---------|---------|---------|-------|
| **本卦** | 全6爻 | 表·全貌剪影 | 外壳、表面、材质、放置、开合 | 体 | 极高(100%) |
| **互卦** | 2-5爻重组 | 里·剖开断面 | 内部结构、内容物、隐藏特征、覆盖机制 | 相-内 | 需交互确认 |
| **变卦** | 动爻推动 | 用·时线去向 | 使用动态、终态、德性归宿 | 相-外/用 | 静卦沉默 |
| **动爻** | 9或6爻 | 位·力点坐标 | 可动部件位置、复杂程度 | 相-精准 | 数=复杂量 |

## 一、本卦——全六爻的气态全貌

### 取爻规则
取全六爻的阴阳排列，不作任何爻位重排。

### 卦脉读法

**核心理念：本卦读取以卦脉德性为纲，不以固定维度框为模板。**
从上下卦的**气脉本质**出发（而非机械填入材质/形态/功能等槽位），自然的特征就会浮现。

```
上卦德性(外/上) + 下卦德性(内/下) + 重卦卦名(全貌)
```

上卦 = 物品的外部/上面/盖子；下卦 = 内部/底部/基座。

### 表达什么

| 特征层 | 读法 | 案例 |
|--------|------|------|
| **外壳形状** | 上卦爻位→上部造型 | 渐䷴巽(柔覆)=盖；大有䷍离(中空)=锁孔 |
| **材质基础** | 德性→材料属性 | 艮=硬质/石；乾=金属/圆 |
| **放置姿势** | 上下关系→摆放方式 | 咸䷞兑上艮下=开口在上、稳定在底 |
| **开合方向** | 动静判断→开启方式 | 贲䷕艮盖住离→外壳闭合遮色 |
| **场态基调** | 卦名→总体状态 | 需=等待；渐=渐进显露；节=周期规律 |

### 确定性：极高（16案例100%命中）
本卦信息不依赖类象库，直接从卦画结构读取。是所有推导的基底。

## 二、互卦——四爻重组的内部断面

### 取爻规则
去掉初爻和上爻，用二三四五爻重组为新卦：
- 下互 = 二三爻组合 → 内层的底层
- 上互 = 四五爻组合 → 内层的上层

### 卦脉读法
互卦的上下两卦不是物品的"上部和下部"，而是**物品内在结构的里层正面和深层**。
互卦读的是——打开外壳之后，内部是什么样？

### 表达什么

| 特征层 | 读法 | 案例 |
|--------|------|------|
| **内部结构** | 空腔/实心/分层 | R1互既济䷾坎中空+离透明→内腔中空透明 |
| **内容物性质** | 细小/颗粒/液态 | R3互姤䷫乾壳+巽细→硬壁装小药丸 |
| **隐藏特征** | 表面看不到的属性 | ④互未济䷿离明+坎暗→包裹内有亮色物 |
| **功能原理** | 运作机制 | R6互夬䷪兑缺口+乾坚实→决断/开启 |
| **里层构造** | 壁/面/层结构 | R5互姤䷫乾壳+巽细→瓷杯壁内容茶 |

### 确定性：中等（需交互确认）
互卦给出的信号在"表里关系"表中属于"里"层，玩家还需通过追问来确认。

### 两条重要规则

1. **每个含巽的互卦 → 优先激活"容纳细小物"关连通道**（姤䷫、益䷩、恒䷟、小蓄䷈、大过䷛等互卦）
2. **🟡互卦信号不得沉默** — 在会合参证被标为中置信后，必须在下一轮追问

### 互卦解读方法：经卦+别卦两层次（v0.7.1-dev）

互卦的解读应与本卦一致，**同时贯通经卦和别卦两个层次**：

```
经卦层：上下互卦分别过月建调节 → 读材质/形态/性质
别卦层：互卦重组为64别卦 → 读内在动态/功效
```

两者协同，缺一不可。

**案例验证**（第8轮·山火贲䷕）：

| 层次 | 读取 | 映射 | 验证 |
|:----|:----|:----|:----:|
| 经卦·坎☵囚 | 流动停止，仅存梗脊脉络 | 茶饼压制纹路 | ✅ |
| 经卦·震☳休 | 活力衰减为纸壳或茶叶 | 纸盒+茶叶 | ✅ |
| 别卦·解䷧ | 破解、散开、分离 | 茶饼/茶砖已掰开 | ✅ |

**核心原则**：别卦名（解=破解）给出内在动态的正解，经卦+月建（坎囚=梗脊、震休=纸壳）给出材质性质的正解。两层次各有承担，合为完整互卦画像。

## 三、变卦——动爻推动的时间去向

### 取爻规则
动爻所在的爻翻转阴阳 → 产生一个新的六爻卦。

### 卦脉读法
变卦 = 物品"使用状态"——不是静止地放着，而是从当前态变化到目标态的过程。

关键判断：**变卦的上卦和下卦各从本卦的什么变成了什么？**

```
例：咸䷞↛蹇䷦
├─ 上卦：兑(开口) → 坎(水/深陷)  → 开口处注入水流
└─ 下卦：艮不变化                 → 杯底不变
→ 注水入杯
```

### 表达什么

| 特征层 | 读法 | 案例 |
|--------|------|------|
| **使用动态** | 德性变化 = 使用过程 | 兑→坎=注水；乾→兑=自完→开口 |
| **终态/结果** | 变卦名 = 使用后的样子 | 渐→谦(盖合上→平铺容器) |
| **德的归宿** | 德的终极归趋 | 变卦震其究=健→大赤色(衣服红色) |
| **功能方向** | 变化从A到B的方向 | 大有→归妹(动+入→插入) |

### 静卦行为
- 无动爻 → 变卦沉默
- 物品没有"使用中的状态"——只是放着。
- 此时只能用错卦反证来补充功能边界

## 四、动爻——力点坐标

### 取爻规则
六爻中数值为9(老阳)或6(老阴)的爻位。

### 表达什么

| 特征层 | 读法 | 案例 |
|--------|------|------|
| **可动部件位置** | 动爻在哪一层 = 哪个层面可变 | 渐动5,6=顶层盖子；咸动4=中层水面 |
| **复杂程度** | 动爻数 = 细节丰富量 | 1动=单一1单功能；3动=间杂/多件 |
| **对应零件数** | 多动 = 多件同类物 | ⑥需→复动3→三把钥匙 |

### 动爻数判断表

| 动爻数 | 射覆对应 |
|--------|---------|
| 0(静卦) | 无变化部件，纯存静态 |
| 1 | 单点活动——一个开合/一处可动 |
| 2 | 两端有活动——上下各一功能件(钥匙的齿+柄) |
| 3+ | 多特征交错——多件同类(多把钥匙)、间杂花纹(衣服方格) |
| 4+ | 气脉突变(天气射覆→忽然起风) |

### 动爻位置判断表

| 动爻位置 | 对应物品部位 |
|---------|-------------|
| 初爻: 最下端 | 底部/插入端 |
| 二爻: 下半中部 | 下体/内容物与下壁交界 |
| 三爻: 正中部(世位) | 核心/中枢/可动中心 |
| 四爻: 上半中部 | 上体/内腔中层 |
| 五爻: 近顶端 | 瓶口/杯口/头部 |
| 六爻: 最上端 | 盖顶/手持端/挂扣 |

## 五、四窗合用——完整度公式

```
物品画像完整度 = 本卦(表) + 互卦(里) + 变卦(用) + 动爻(位)

静卦时：物品画像完整度 = 本卦(表+里合一) + 互卦(本卦自身就是互) + 错卦反证 + 动爻(无)
                                                                              → 静卦时用错卦替代变卦的功能边界
```

## 六、快速判定流程

```
起卦
  ↓
0a-e 卦脉初判（内外卦德性+下互+上互每卦必经）
  ↓
0f 月建加权（上下卦旺衰→五维度系数调节）
  ↓
本卦：输出 外壳·材质·场态·开合    【表·体】
  ↓
互卦：输出 内部结构·内容物·功效    【里·相-内】 ← 经卦分别受月建调节
  ↓
变卦：输出 使用动态·终态·德归趋    【用·相-外】（静卦则用错卦）
  ↓
动爻：输出 可动位置·复杂度          【位·相-精度】
  ↓
双卦参证 → 特征画像v1 → 玩家交互 → 渐进精化
```

## 七、16案例命中率同四窗质量的关系

| 命中率范围 | 互卦质量 | 变卦质量 | 代表性案例 |
|-----------|---------|---------|-----------|
| 83-90% | ✅全开 | ✅精准 | ①聪哥衣服、⑨眼镜 |
| 70-83% | ✅主体捕中 | 🟡部分沉默 | ④戒指、⑥钥匙、⑦碗筷 |
| 20-40% | 🔴关键窗口沉默 | 🔴信号偏调 | ⑪打火机、⑫充电头、⑬耳塞 |
| 0-7/7 | ✅全开 | ✅精准(静卦用错卦) | R2酸枣仁、R3药瓶 |

**核心结论**：命中率 = 四窗打开的完整度。窗口沉默越多 → 命中越差。
