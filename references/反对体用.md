## 一、反对三义——玄明的总纲框架

> 玄明：「卦反之义 = 同一事的主客两面（本余、气之来去、时间先后）；卦对之义 = 同一气的阴阳两态（对待、互根互用、同时共存）。反对交织，方见一事之全、一气之整。」

### 1.1 反体——主客翻转，本余相次

| 维度 | 含义 |
|---|---|
| **操作** | 卦画上下倒转 |
| **气** | 气的来去相反 |
| **视角** | 主客翻转——主为本气（阳，施也），客为余气（阴，受也） |
| **时序** | 本先余后——主人先到场，宾客后到来。这是气的时序，非人力安排 |

### 1.2 对体——阴阳全反，互根互用

| 维度 | 含义 |
|---|---|
| **操作** | 每爻阴阳互换 |
| **气** | 同一造化之气的呼吸两端 |
| **关系** | 不是外在对立（高vs低），而是互为前提——乾之施与坤之受同时而在 |
| **时序** | 无时间先后——同时共存 |

### 1.3 反与对的根本区别

| | 反体（主客） | 对体（对待） |
|---|---|---|
| **关系** | 同一事，视角翻转 | 同一气，阴阳全反 |
| **逻辑** | 本气与余气——先主后客 | 对待与流行——互根互用 |
| **合一方式** | 主客合为事的全体 | 阴阳合为气的整体 |
| **时间性** | 有先后——本先余后 | 无先后——同时共存 |
| **维度** | **时** | **位** |

> 玄明：「卦反是"时"的关系，卦对是"位"的关系。」

## 二、反对关系的三种形态（20 组全）

> 玄明分类体系

### 2.1 自反对体 4 对（8 卦）——反体归己，唯对成双

| # | 对体对 | 核心义 | 何以自反 |
|---|---|---|---|
| 1 | 乾 ☰☰ ↔ 坤 ☷☷ | **天地之极**：施气之源与受气之场——同一造化的呼与吸 | 天无客（天之上更无天），地无客（地之下更无地） |
| 2 | 坎 ☵☵ ↔ 离 ☲☲ | **造化之枢**：内润之功与外明之效——润是明的内功，明是润的外效 | 枢机自还，不假外转 |
| 3 | 颐 ☳☶ ↔ 大过 ☴☱ | **养与行**：养正于内与过任于外——养而不行则腐，行而不养则折 | 结构对称于中轴 |
| 4 | 中孚 ☱☴ ↔ 小过 ☶☳ | **信与行**：虚中致信与中实行过——信而不行则虚，行而不信则妄 | 皆以中为主，自身即全体 |

**Skill 处理**：这些卦的对体即是自身（反体）。在推演中标注「此卦无主客之分——自身即是全体」。对体分析聚焦于其对待卦的互根关系。

### 2.2 反=对 4 对（8 卦）——时位合一

| # | 对子 | 核心义 | 时位合一的含义 |
|---|---|---|---|
| 5 | 泰 ↔ 否 | **交与不交**——无中立地带 | 同一天地，交则为泰，不交则为否。翻转视角即是阴阳全反 |
| 6 | 随 ↔ 蛊 | **从与败**——悦极则惰，惰则生蛊 | 同一次追随，追随时是随，追随久了成蛊 |
| 7 | 渐 ↔ 归妹 | **缓与急**——正道与权变 | 同一前行，缓步是渐，急行是归妹 |
| 8 | 既济 ↔ 未济 | **成与未成**——成则向败，败则求成 | 同一事业，成时为既济，未成时为未济 |

**Skill 处理**：这是反对关系中最紧密的一种——主客翻转即阴阳全反。推演中强调「此二卦之间无缓冲、无过渡——翻转即是全变」。

## 三、从卦脉到反对——宏观与微观的统一

> 玄明：「单卦观脉——气在一卦之内如何流行（微观）。反对观网——气在诸卦之间如何来去、如何对待（宏观）。反对是卦脉的横向延伸，卦脉是反对的纵向基础。」

### 3.1 B1 与 B2 的关系

```
B1 卦脉（单卦）：六爻内 → 爻位即气位 → 观一卦之脉
B2 反对（诸卦）：卦之间 → 反对即气之来去对待 → 观诸卦之网
```

反对不离卦脉。不知师之脉（阳陷地中），便不知师何以为主将帅众；不知比之脉（水流地上），便不知比何以为众人相比。**每一对反对关系的解读，必须以单卦的卦脉为基础。**

### 3.2 反对分析中融入卦脉

以需 ↔ 讼为例：

| 步骤 | 卦脉分析 | 反对解读 |
|---|---|---|
| 需之卦脉 | 坎水在乾天之上——水在天，气自上而下施 | **主家视角** |
| 讼之卦脉 | 乾天在坎水之上——天上行水下行，气自下而上索 | **宾客视角** |
| 反对关系 | 同一宴席，气从相反方向来 | 主见施，客见争——合为宴享之全 |

## 四、反对分析的 Skill 操作框架

### 4.1 新增步骤：Layer 4.6 反对分析

在 Layer 4.5（卦画默观）之后：

```
Layer 4.6 反对分析（新增）
  │
  ├── 4.6.1 判定反对形态
  │   本卦属于哪一类？
  │   · 自反对体（4对8卦）——反体归己，无主客
  │   · 反=对（4对8卦）——时位合一
  │   · 标准四卦组（12组48卦）——反对交织
  │
  ├── 4.6.2 反体（综卦）分析
  │   若是自反：标注「无主客之分，自身即全体」
  │   若有反体：卦脉解读反体卦 → 主客翻转后气的来去
  │   → 得出"同一事，从另一边看呈现什么态势"
  │
  ├── 4.6.3 对体（错卦）分析
  │   解读对体卦的卦脉
  │   → 得出"同一气，阴阳全反后呈现什么互根关系"
  │
  └── 4.6.4 四维合观（仅标准四卦组）
      A之反体=B（主客） + A之对体=C（对待）
      → 得出四卦交织的全图
      → "这件事的完整图景：A→B→C→D"
```

### 4.2 反对分析的语言模板

**反体（主客）**：
```
🔄 反体视角（综卦：XXX）
本卦卦脉：……（主家视角）
反体卦脉：……（宾客视角）
同一件事，站的位置不同，看到的景象不同。
在当前事项中，这意味着：……
```

**对体（对待）**：
```
🔍 对体视角（错卦：XXX）
本卦之气：……
对体之气：……
这不是两个对立面，而是同一气的呼吸两端——互为前提、同时共存。
在当前事项中，这意味着：……
```

**四维合观**（标准四卦组）：
```
🔗 四维合观
A(本卦)→B(反体)：主客翻转后的景象
A(本卦)→C(对体)：阴阳全反后的互根关系
四卦合为：……（全图）
```

### 4.3 自反对体的特殊处理

```
⚠️ 此卦为自反卦——反体即自身。这意味着：
此卦无主客之分，自身即是全体。气的动力来自内部，不依赖外部对比。
对体卦（XXX）是其对待——二者是同一气的呼吸两端。
```

### 4.4 反=对的特殊处理

```
⚡ 此卦的反体 = 对体 ——时位合一。
主客翻转即是阴阳全反。二卦之间无缓冲、无过渡。
这意味着在当前事项中，"……"与"……"之间没有中间态——翻转即是全变。
```

## 五、反对与体用的关系

### 5.1 定位

反对分析**不与体用生克冲突**，是增加的维度：

```
体用分析：我 vs 对方（当下的力量对比）    ← 现有
反对分析：主 vs 客 + 阳 vs 阴（时空展开）  ← 新增
```

### 5.2 B1→B2 的推演堆叠

```
B1 卦脉（单卦内） → 本卦的德性与落处
  │
  ├→ 体用生克（本卦的上下卦关系）    ← 五行层（后续 Phase）
  │
  └→ B2 反对（诸卦间） → 主客翻转 + 阴阳对待
       │
       ├→ 反体：时间的反转
       ├→ 对体：性质的互根
       └→ 四维合观：事的全图
```
