# B5 变复求义——一卦八体与气脉校准

> 基于互体参变篇 + 乾坤真体研读 + 乾宫八变 + 变复验证
> 目标：为解卦 Skill 建立「自校验」机制——推演的结果对不对，用变复来验

---

## 一、B5 的独特定位

B1 教怎么看（符号观），B2 教怎么比（反对），B3 教怎么流（进退），B4 教怎么翻（落处）。

B5 教的是：**你怎么知道自己看得对不对？**

> 玄明：「卦义未审，须求变复。不唯辞合，义实通明。」——当卦义不清楚或不确定时，用变复来验证。变是看前后帧，复是看横轴互补帧。看了前后左右，这一帧的意思自然清楚。

B5 = 解卦 Skill 的**自校验层**。

## 二、一卦八体——本卦的八个视角

> §19：「一卦之中，凡具八卦。有正有伏，有互有参。」

### 2.1 八体定义

| 体 | 操作 | 屯卦为例 | 揭示什么 |
|---|---|---|---|
| **本卦** | 六爻不动 | 屯 ☵☳ | 当前定格 |
| **正·下体** | 取内卦三爻 | 震 ☳ | 内在根基的德性 |
| **正·上体** | 取外卦三爻 | 坎 ☵ | 外部环境的德性 |
| **上伏** | 上体变对体，下体不动 | 离上震下 = 噬嗑 ☲☳ | 外部条件变化时的偏向（外感之势） |
| **下伏** | 下体变对体，上体不动 | 坎上巽下 = 井 ☵☴ | 内在素质变化时的偏向（内生之势） |
| **下互** | 二三四爻成卦 | 坤 ☷ | 腹部下层脏腑 |
| **上互** | 三四五爻成卦 | 艮 ☶ | 腹部上层脏腑 |
| **参** | 下互+上互合为六画卦 | 艮上坤下 = 剥 ☶☷ | 脏腑整体的完整格局 |

### 2.2 伏的精确含义

> 玄明：「伏不是对体。对体是六爻全变（全变），伏是偏变——只变一极，留一极不动。伏是介于本卦与对体之间的临界状态。」

| | 对体（B2） | 伏（B5） |
|---|---|---|
| 操作 | 六爻全变 | 只变上体或下体 |
| 结果 | 全盘反转 | 局部偏向 |
| 揭示 | 气的完整对岸 | 气正在变的势 |

**上伏**：外部（上体）变成对体 → "如果外部条件翻转，会偏向哪个方向？"
**下伏**：内部（下体）变成对体 → "如果内部根基翻转，会偏向哪个方向？"

### 2.3 互体与中爻

> §25：「中爻之义，足为造化。」陈抟：「二三四五，造化之气，参互成卦。」

互体取中四爻——初爻是始、上爻是终，而**中间才是气正在运行的区域**。互体揭示的是「在表面态势之下，腹部正在酝酿什么」。

### 2.4 参——脏腑合为格局

以上互为上卦、下互为下卦组成的新六画卦——**参卦揭示了本卦内部脏腑构成的完整格局**。

陈抟举例：「屯中有剥，蒙中有复。」屯的参卦是剥——始生之难的腹部藏着阴消阳的剥势。蒙的参卦是复——蒙昧求明的腹部藏着一阳来复的生机。

## 三、变复验证——气脉校准器

> 玄明：「变是看前一帧或后一帧，复是看横轴上的互补帧。看了前后左右，这一帧的意思自然清楚。」

### 3.1 变的操作（一爻之变）

```
操作：改变本卦中某一爻的阴阳，看变成什么卦
揭示：阴阳二气在单一位置上的消长——同一位置，气态翻转后是什么

法则：
· 动爻所在位置——优先对这个位置做变（因为是当前变化的焦点）
· 如果没有指定动爻，逐爻变之，看各有何走向
```

**例**：蒙卦上九阳爻 → 变为师卦上六阴爻

| | 蒙上九 | 师上六 |
|---|---|---|
| 爻辞 | "击蒙"——冲破蒙昧 | "小人勿用"——堆叠之势不可动 |
| 气态 | 阳爻冲破重重阴蒙 | 阴爻积压堆叠 |
| 验证 | 蒙的意义确实与"冲破蒙蔽"有关——因为同一位置气态翻转后是"堆叠不动"，二者正好构成"破/积"的互补 | ✅ 通明 |

> 玄明三原则之一：「气位非道德」——「小人勿用」中的「小人」不是道德贬义，是气位关系的客观描述：阴爻位置低、阴气积压堆叠，故「勿用」=不可轻易动作。

### 3.2 复的操作（一体之复 = 伏）

```
操作：将上体或下体翻转为对体（即伏的操作），从一体之变看卦义
揭示：乾坤二气在整体结构中的呼应——一侧翻转后，另一侧的补位

与变的区别：
· 变 = 爻层面（微观，一爻动）
· 复 = 体层面（宏观，一体翻）
```

**例**：大有卦

| 复操作 | 结果 | 验证 |
|---|---|---|
| 上体离 → 复为坎 | 坎在离的位置 → 收纳解发放 | 得需"有饮食之宴"——大有之义含"能放能收" ✅ |
| 下体乾 → 复为坤 | 坤在乾的位置 → 虚中受纳 | 得晋"有昭明之德"——大有之义含"光照虚中" ✅ |

### 3.3 三原则

> 玄明确认的三条方法论原则：

**原则 1：气位非道德**
卦爻中的"贵贱""小人""吉凶"等辞，皆是气位关系的客观描述，无涉道德褒贬。阴爻在低处是"卑"——这是位置描述，不是人格贬低。

**原则 2：对待中见互补**
每一卦都包含其对岸：离中已含坎之机（中爻为阴，阴虚能受水），乾中已含坤之质（纯阳的边界就是阴的起点）。复的操作是把潜藏的对岸显化出来，从而验证本卦之义。

**原则 3：辞合不如义通**
验证卦义时，不追求卦爻辞的文字表面吻合（"不唯辞合"），而是看气脉是否贯通（"义实通明"）。如果气脉贯通，辞不合也可以；如果气脉不通，辞合也是牵强。

## 四、Skill 操作框架

### 4.1 新增步骤：Layer 6b 变复校验

在 Layer 6a（基础综合推演）之后，输出最终报告之前：

```
Layer 6b 变复校验（新增）
  │
  ├── 6b.1 八体展开（可选，按需触发）
  │   如果用户要求深入 / 推演结果与预期差异大 / B3 判定"表里不一致" →
  │   展开一卦八体：正(2体) + 伏(2卦) + 互(2体) + 参(1卦)
  │   每个体简要标注其德性与关联
  │
  ├── 6b.2 变复验证（默认触发）
  │   ├── 爻变验证（如有动爻）：
  │   │   动爻阴阳翻转 → 变出何卦 → 变卦与本卦的气态对比
  │   │   → "同一位置，气态翻转后是……，说明当前气态的态势是……"
  │   │
  │   └── 体复验证（默认）：
  │       上体复为对体 → 上伏卦 → 外部条件变化后的偏向
  │       下体复为对体 → 下伏卦 → 内部根基变化后的偏向
  │       → "从伏卦回看本卦，验证本卦解读的合理性"
  │
  ├── 6b.3 参卦造化判定
  │   互体参合的卦是什么？
  │   参卦与本卦是否一致？
  │   → 一致：表里如一，当前解读可信度较高
  │   → 不一致：存在祸福倚伏，需标注"表面态势与内部造化存在差异"
  │
  └── 6b.4 校验结论
      本次推演经变复验证：通明 / 存疑（标注存疑点）
```

### 4.2 何时触发八体全展开 vs 仅变复验证

| 场景 | 触发级别 |
|---|---|
| 用户首次推演、事项不复杂 | 仅变复验证（爻变 + 上下体复） |
| B3 判定"表里不一致"（祸福倚伏） | 八体全展开 |
| 用户要求深入分析 | 八体全展开 |
| 用户反馈推演与实际情况不符 | 八体全展开 + 逐爻变验证 |

### 4.3 校验结论的语言模板

```
🔬 气脉校准
爻变验证（动爻位置）：……
  同一位置气态翻转后为……，验证当前气态的……属性。

体复验证：
  上伏（外部变）：……
  下伏（内部变）：……
  从伏卦回看，本卦的……解读得到印证 / 需要调整……

参卦造化：参卦为 [卦名]，……
  与本卦态势 [一致/不一致] → [表里如一/存在倚伏]

校验结论：本次推演经变复验证，气脉 [通明/部分通明/存疑]
  [如存疑，标注具体存疑点]
```

## 六、B5 小结

| 维度 | 核心 |
|---|---|
| 一卦八体 | 本卦 + 正(2) + 伏(2) + 互(2) + 参(1) = 从八个视角看同一个卦 |
| 伏≠对体 | 伏是偏变（一端翻），对体是全变（全翻）。伏揭示"正在变的势" |
| 变（一爻之变） | 微观校验——同一位置气态翻转后的对比 |
| 复（一体之复） | 宏观校验——外部或内部翻转后的偏向 |
| 参（互体合卦） | 造化校验——腹部脏腑构成的完整格局 |
| 三原则 | 气位非道德 / 对待中见互补 / 辞合不如义通 |
| 触发策略 | 默认做变复验证；表里不一致或用户要求时才全展开八体 |
