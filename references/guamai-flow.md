# B3 卦脉流动——六爻内的阴阳进退

> 基于卦脉篇 + §25「中爻之义，足为造化」+ 乾宫八变 + 余气篇的阴阳进退原理
> B1 教怎么看单卦德性，B2 教卦与卦的关系，B3 教六爻之内气的流动轨迹

---

## 一、从卦脉到卦内流动

B1 的「爻位即气位」告诉我们在哪个爻位阳气在哪个爻位阴气在——这是静态的「位」。

B3 要回答的是动态的「流」：**从初爻到上爻，气是怎么走的？进还是退？通还是塞？**

> 玄明札记#1：「爻位即气位——阳爻在哪里，阳气就在哪里；阴爻在哪里，阴气就在哪里。气的运行轨迹与节奏，就是卦脉。」

「轨迹」和「节奏」——这就是 B3 要量化的东西。

## 二、消息盈虚——阴阳进退的总模板

### 2.1 十二消息卦

六十四卦中有十二卦是阴阳进退的「标准刻度」：

```
阳息（阳长阴消）：
复 ☷☳ → 临 ☷☱ → 泰 ☰☷ → 大壮 ☰☳ → 夬 ☱☰ → 乾 ☰☰
1阳       2阳        3阳        4阳         5阳       6阳

阴消（阴长阳消）：
姤 ☰☴ → 遁 ☰☶ → 否 ☷☰ → 观 ☴☷ → 剥 ☶☷ → 坤 ☷☷
1阴       2阴        3阴        4阴         5阴       6阴
```

**规律**：阳气从初爻开始，逐爻向上积累——从底部萌动（复），到中间均衡（泰），到顶部全满（乾）。然后阴气从初爻开始同样路径——到底部全满（坤），再循环回阳气初生（复）。

### 2.2 任意卦的阴阳进退判定

拿到任意六爻卦，先不看卦名，只看阴阳分布：

```
判定法则：
1. 数阳爻个数 → 定位在 0~6 阳的哪个阶段
2. 看阳爻的位置分布 → 阳气是集中在下（初发）、中（运行）、还是上（将尽）？
3. 看阴阳的交错模式 → 是阳逐步被阴逼退？还是阴逐步被阳冲破？
```

**四种基本进退模式**：

| 模式 | 特征 | 实例 | 卦脉解读 |
|---|---|---|---|
| **阳进** | 阳爻集中在下方，阴爻在上方 | 泰 ☰☷（3阳下 3阴上） | 阳气自下而上推进——事在上升期 |
| **阳退** | 阳爻集中在上方，阴爻在下方 | 否 ☷☰（3阴下 3阳上） | 阳气被阴逼退至顶端——事在衰退期 |
| **阴进** | 阴爻集中在下方，阳爻在上方 | 姤 ☰☴（1阴在下 5阳在上） | 阴气从根部渗入——隐患在暗中滋生 |
| **阴退** | 阴爻集中在上方，阳爻在下方 | 复 ☷☳（5阴在上 1阳在下） | 阴气被阳气从底部突破——新机萌动 |

### 2.3 混交卦的复杂进退

不是所有卦都像消息卦那样整齐。大多数卦的阴阳是交错的。对于交错卦，判定方法：

```
交错判定：
1. 初爻是阳还是阴？→ 气的起点性质
2. 上爻是阳还是阴？→ 气的终点性质
3. 从初到上，阳爻的比例是升还是降？→ 整体走向
4. 哪一段的阴阳交错最密集？→ 气的激荡区
```

**示例**：屯卦 ☵☳（阳、阴、阴、阴、阳、阴）

- 初爻阳 → 起点有阳气萌动
- 上爻阴 → 终点被阴气覆盖
- 中间 2-4 爻全阴 → 阳气进入后立即被阴包围（陷）
- 整体：阳初动→被阴吞没→阳再出（五爻阳）→最终被阴封顶

**卦脉解读**：屯是「始生之难」——阳气突破了两层（初阳+五阳），但在突破之间被三层阴气压住。初生之物，出而受阻，再出再阻——这就是「屯难」的气脉。

## 三、中爻造化——二至五爻的生成区

> §25：「中爻之义，足为造化。纳音切脚，其理则一。」
> 陈抟：「二三四五，造化之气，参互成卦。」

### 3.1 六爻的三区划分

```
上爻 ── 气之终（完成区）
五爻 ──┐
四爻 ──┤ 中爻 = 造化区（气正在运行、正在生成的区域）
三爻 ──┤
二爻 ──┘
初爻 ── 气之始（发动区）
```

- 初爻：气的起点——一件事的「第一推动」
- 上爻：气的终点——一件事的「最后结果」
- 中四爻：气正在运行——**祸福倚伏都发生在这里**

### 3.2 互体揭示了什么

B5 会详细展开互体参变。但在 B3 的流动分析中，互体的核心功能是：

> 玄明互体篇：「互体是藏在卦的腹部的体——不在表面，而在中间。如同脏腑。」

**互体取中四爻，正因此处是造化发生的场所。** 互体揭示的是：在表面可见的上下卦结构之下，中间的气正在酝酿着什么。

**实用法则**：
```
本卦的表面态势 = 上下卦的德性关系（B1）
本卦的内部造化 = 互体的德性关系（B3）
二者一致 → 表里一致，态势稳定
二者不一致 → 表面一套，内部一套——祸福倚伏
```

**屯卦例**：
- 表面：坎上震下 → 外险内动（艰难中行动）
- 内部：坤为下互、艮为上互 → 厚重承载被止定封顶（内部已有剥的种子）
- 不一致 → 表面在创业，内部已在消耗——「生中藏消」

## 四、卦脉流动的 Skill 操作框架

### 4.1 新增步骤：Layer 4.7 卦脉流动分析

在 Layer 4.6（反对分析）之后：

```
Layer 4.7 卦脉流动分析（新增）
  │
  ├── 4.7.1 阴阳进退判定
  │   本卦几阳几阴？
  │   阳爻主要分布在哪一段？（下/中/上）
  │   属于哪种进退模式？（阳进/阳退/阴进/阴退/交错）
  │
  ├── 4.7.2 消息卦参照
  │   本卦的阴阳分布最接近哪个消息卦？
  │   → 定位在阴阳循环的大周期中
  │
  ├── 4.7.3 中爻造化分析
  │   下互体 + 上互体的德性
  │   互体德性与本卦上下卦德性是否一致？
  │   → 一致 = 表里如一；不一致 = 祸福倚伏
  │
  └── 4.7.4 流动初判
      从初爻到上爻，气在做什么？
      关键爻位（初爻的启动 + 互体的造化 + 上爻的归宿）
      整体态势：进 / 退 / 通 / 塞 / 激荡 / 沉滞
```

### 4.2 流动分析的语言模板

```

初爻（起点）：……（阳/阴，什么德性）
中爻（造化）：……（互体显示了什么内部动态）
   本卦表面是……，内部互体是……——→ [一致/不一致]
上爻（终点）：……（阳/阴，什么归宿）
整体走向：……（进/退/通/塞/激荡/沉滞）
消息定位：本卦最接近 [消息卦名]，处于阴阳循环的 [上升/下降] 段
```

## 五、B3 小结

| 维度 | 教什么 |
|---|---|
| 阴阳进退 | 从初→上的阳气/阴气比例变化 → 四种进退模式 |
| 消息定位 | 最接近哪个消息卦 → 在阴阳大周期中的位置 |
| 中爻造化 | 互体是「腹部的脏腑」→ 表里是否一致 |
| 祸福倚伏 | 表里不一致 → 表面态势下藏着另一种气脉 |

**B3 的本质**：B1 看「每个爻位是什么气」，B3 看「这些气从初到上是怎样流动的」。B1 给的是快照，B3 给的是动画。
