## S1 正易心法五原则

### 原则一：爻位即气位
阳爻在哪里，阳气就在哪里；阴爻在哪里，阴气就在哪里。**不看标签，看位置。**

### 原则二：八卦是八种功能态
八卦不是名词，是德性（功能态）。从卦画结构直接读：

| 乾 | 坤 | 震 | 巽 | 坎 | 离 | 艮 | 兑 |
|---|---|---|---|---|---|---|---|
| 三阳 | 三阴 | 一阳**下** | 一阴**下** | 一阳**中** | 一阴**中** | 一阳**上** | 一阴**上** |
| **健** | **顺** | **动** | **入** | **陷** | **丽** | **止** | **悦** |
| 纯阳充盈 | 纯阴顺承 | 阳在阴底 | 阴在阳基 | 阳陷阴中 | 阴附阳中 | 阳冒阴顶 | 阴达表面 |
| 健动不息 | 空无静待 | 涌动冲击 | 渗透滋长 | 在夹缝流 | 借阳四射 | 涌无可涌 | 满溢四流 |

### 卦脉总纲细则

每卦的气脉定义（正易心法·陈抟注）是卦象分析的第一性原理：

- **震☳**：阳在重重阴气之下、之内、之底，涌动冲击。阳被阴压，蓄力爆发。德性：动。身体：足。
- **巽☴**：阴滋长于阳之内部、阳之根底，渗透滋长。阴从根基渗入阳中。德性：入。身体：手。
- **坎☵**：阳陷于阴质之围绕包夹，在阴质包夹中流行。阳被阴困于核心，被迫流动。德性：陷。身体：耳。
- **离☲**：阴附丽于阳中，借阳而四射。阴依附于阳而发光，借势发散。德性：丽。身体：目。
- **艮☶**：阳上冒于阴之顶，涌无可涌，极止（不离阴也）。阳涌到顶端自然停止。德性：止。身体：鼻。
- **兑☱**：阴滋长到达于上、于外、于表，满溢四流。阴从内部满溢至表面。德性：悦。身体：口。
- **乾☰**：纯阳充盈，无阴杂，圆满健动不息。纯粹动能→自完自足运行。德性：健。无阴故无具体身体对应。
- **坤☷**：纯阴承载，无阳动，空无静待。纯粹形质→承阳而顺。德性：顺。无阳故无具体身体对应。

> 重要纠正（以正易心法为准，传世《说卦传》有误）：艮为鼻（非手），巽为手（非股）。
> 六子卦皆阴阳不相离——唯一的异类爻嵌在同类爻包围之中，阴阳互相依存。

### 原则三：观不是推
先静观卦画结构——六爻阴阳怎么排列，气就怎么走。**观是直观照见，不是逻辑推导。**

### 原则四：同构容纳
同一气脉可落于多个维度——自然、人身、器物、人伦。**一脉多现。**

### 原则五：合于自然者为正
每个推演必须能从卦画结构回溯验证。方向不可逆：**结构 → 自然 → 语境**，不能从语境倒推。

## S3 射覆分析体系

核心方法：卦脉初判 → 六十四卦气脉德性分析 → 四类规则展开 → 两阶段交互。

> 射覆仅限物品特征画像。禁止用于事业/天气/金融等非射覆场景预测。

---

### 3.0 卦脉初判（每卦必经前置）

**0a. 读内卦（下卦）气脉** — 从六子卦气脉定义中读取内卦气形态
**0b. 读外卦（上卦）气脉** — 同上
**0c. 定卦德** — 上下卦德性复合 → 别卦整体性质
**0d. 判气象** — 全卦阴阳数、包围关系、气机通塞（通/塞/交/隔/聚/散/冲/纳）
**0e. 开卦象** — 以卦脉校验展开合法类象，排除不符气脉者
**0f. 月建加权** — 判定旺相休囚死 → 五维系数调节

### 3.1 六十四卦气脉德性分析

以 `references/64卦卦脉全解.md` 为底，一口气读完本卦整体：
- 上下卦德性复合 → 别卦整体性质
- 自然落处参照系（仅作参照，非推断落处）
- 卦名本质含义（非词义，是气的态势的浓缩表达）

→ 产出：别卦整体气脉德性画像（自然参照系+核心态势）

### 3.2 四类规则展开

参见 `references/射覆四类卦脉应用规则.md`。四个窗口各自有职责边界，不互替代：

| 窗口 | 取爻 | 表达 | 对应维度 | 确定性 |
|------|------|------|---------|:------:|
| **本卦** | 全6爻 | 外壳、表面、材质、放置、开合 | 材质/形态/放置/开合 | 极高 |
| **互卦** | 2-5爻重组 | 内部结构、内容物、隐藏特征 | 结构/内容物/隐藏层 | 需交互 |
| **变卦** | 动爻推动 | 使用动态、终态、德性归宿 | 使用方式/终态 | 静卦沉默 |
| **动爻** | 9或6爻 | 可动件位置、复杂程度 | 可动点/变化部位 | 数=复杂度 |

### 3.3 Phase 1：特征画像（不提供品名）

```
步0卦脉初判 + 3.1别卦气脉 + 3.2四类规则 + 0f月建系数

→ 特征画像 v1：
  仅输出：材质 🟡 / 形态 🟡 / 放置 🟡 / 开合 🟡
  所有维度初始🟡，禁止🟢锁定
  
  ⛔ 禁止：品名、种类、候选列表、用途揣测
```

**交互门**：v1 产出后，玩家反馈「对了/偏了/XX不对」。本卦维度确认后 🟡→🟢 锁定。

```
v2 产出（本卦确认后）：
  材质 🟢 / 形态 🟢 / 放置 🟢 / 开合 🟢 (锁定)
  结构 🟡 / 内容物 🟡 / 隐藏特征 🟡 (互卦)
  
v3 产出（互卦确认后）：
  所有已确认 🟢 锁定
  使用动态 🟡 / 终态 🟡 (变卦)
  可动点 🟡 (动爻)
```

上限 5 轮交互。玩家说「够了」则跳过剩余→直入 Phase 2。

### 3.5 交叉校验 + 学习保存

校验规则：
- 四类窗口相互印证 → 🟢 高置信
- 窗口间矛盾 → 以本卦窗口为准（确定性最高）
- 0e类象校验 → 品名必须在合法列表中
- 0f月建系数调节 → 每维度以系数调描述语气

每轮结束后写入 `records/pending_updates/{日期}_{序号}.json`，不自动更新数据。
需要人工验证后才合并入正式数据（类象库/月建映射/四类规则/用户聚类）。

## S4 落处体系

### 射覆落处（物性维度·仅此一个）

将气态投射为物品特征：材质 / 形态 / 放置 / 开合（本卦）→ 结构 / 内容物 / 隐藏特征（互卦）→ 使用动态 / 终态（变卦）→ 可动点（动爻）。

每维度输出特征描述 + 卦象依据 + 置信度 🟢🟡🔴。

问事落处已从此版本移除——射覆 Skill 仅限射覆物品。

## S5 评估标准

核心：评估的是特征刻画精度，不是命名准确率。AI 是画师不是鉴定师。

### 两阶段评估
- **Phase 1 评估**：特征命中率 = 本卦/互卦/变卦/动爻各窗口维度命中数
- **Phase 2 评估**：在 Phase 1 及格基础上，品类正确 + 名称在选项中

### 学习反馈
- 不自动更新。全部写入 pending_updates，人工验证后方合并。
- 验证入口：`python scripts/update_kb.py --review-pending`
