# 射覆 - 正易心法卦脉解读 Skill

> 基于麻衣道者《正易心法》42 章·陈抟注的卦脉解读方法
> 不查表、不套口诀——从卦画结构中直接读取气的态势

---

## 1. 核心原则

### 原则 1：爻位即气位

阳爻在哪里，阳气就在哪里；阴爻在哪里，阴气就在哪里。**不看标签，看位置。**

### 原则 2：八卦是德性，不是名词

八卦 = 8 种功能态（健顺动入陷丽止说），从卦画结构直接读出：

| ☰ 乾 | ☷ 坤 | ☳ 震 | ☴ 巽 | ☵ 坎 | ☲ 离 | ☶ 艮 | ☱ 兑 |
|---|---|---|---|---|---|---|---|
| 三阳纯奇 | 三阴纯偶 | 一阳在下 | 一阴在下 | 一阳在中 | 一阴在中 | 一阳在上 | 一阴在上 |
| **健** | **顺** | **动** | **入** | **陷** | **丽** | **止** | **说/悦** |

### 原则 3：观不是推

先静观卦画结构（六爻阴阳排列），再做事项语境的翻译。观是直观照见，不是逻辑推导链条。

### 原则 4：同构容纳

"包括万象"的机制是同构——卦脉结构在自然、人身、人伦、治理维度有同构落处。**同一气脉，一脉多现。**

### 原则 5：合于自然者为正

每个推演判断必须能从卦画结构回溯验证。方向不可逆：**结构 → 自然 → 语境**，不能从语境倒推。

---

## 2. 分析工作流

### 输入格式

使用前准备：
- 专注于被覆盖隐藏的物体，内心生问"对方藏的是什么？""这是什么？"
- **暂不提供起卦功能**，需自行提供卦名和月、日干支。

```
卦名：[本卦名]（用户自行提供）
月干支：[如 癸巳]
日干支：[如 戊戌]
变卦：[如有变动，给出变卦名]
动爻：[如有，给出爻位 1-6]
事项：[自然语言描述所问之物]
```

### Layer 0：信息确认

若输入不完整（缺卦名/事项），先引导补充。确认格式后进入分析。

### Layer 1：卦画默观（B1）

**不看卦名，不看卦辞，只看六爻阴阳排列。**

```
1. 画出六爻阴阳分布（从初到上）
2. 上卦三爻 → 什么德性？（健/顺/动/入/陷/丽/止/说）
3. 下卦三爻 → 什么德性？
4. 上下卦德性关系？（上压下？下托上？对抗？配合？）
5. 整体初判：这个卦的态势方向是进/退/困/通/散/聚？
```

然后参照对应的《说卦传》第 11 章类象展开落处，但**先做结构默观，后查类象验证**。

### Layer 2：反对分析（B2）

```
反体（综卦）—— 同一事，从另一边看（主客翻转，时间维度）
  卦画上下倒转 → 得到什么卦？
  → "同一组气脉，如果终点回看起点，呈现为什么态势？"

对体（错卦）—— 同一气，阴阳全反（对待互根，位维度）
  每爻阴阳翻转 → 得到什么卦？
  → "如果每一爻都反转，完全对立的可能性是什么？"

判定反对形态：
  · 自反对体（乾坤坎离颐大过中孚小过）→ 反体即自身，无主客
  · 反=对（泰否/随蛊/渐归妹/既济未济）→ 时位合一，无缓冲
  · 标准四卦组（其余48卦）→ 四维合观，展开全图
```

参考 `references/反对体用.md` 获取 20 组反对关系的完整映射。

### Layer 3：卦脉流动（B3）

```
阴阳进退判定：
  几阳几阴？阳爻主要分布在哪一段（下/中/上）？
  → 阳进 / 阳退 / 阴进 / 阴退 / 混交

消息卦参照：
  本卦最接近哪个消息卦（复临泰大壮夬乾/姤遁否观剥坤）？
  → 定位在阴阳大周期中的位置

中爻造化（互体）：
  下互体（二三四爻）的德性？
  上互体（三四五爻）的德性？
  → 表里是否一致？
  → 一致 = 表里如一；不一致 = 祸福倚伏
```

### Layer 4：月建加权（补充层）

```
获取月建 → 查上下卦五行 → 旺相休囚死判定：

  八卦五行：乾兑=金，震巽=木，坎=水，离=火，坤艮=土
  月建五行：寅卯=木，巳午=火，申酉=金，亥子=水，辰戌丑未=土

  旺：卦五行 = 月建五行（同气当令）
  相：月建生卦（得令生扶）
  休：卦生月建（气泄于令）
  囚：卦克月建（犯令被制）
  死：月建克卦（被令所克）

月建不改卦脉之质，只改卦气之量（强弱/持久/短暂）。
```

### Layer 5：落处展开（B4）

将卦脉结构翻译到用户事项语境：

```
1. 提炼核心卦脉模式（1-2句话）
   例："内在纯阳驱动（乾健），外部遭遇夹击困境（坎陷）"

2. 多维落处展开：
   · 自然维度（1条，作为参照系）
   · 用户事项维度（2-3条，核心）
   · 辅助维度（1条，精选）
   每个落处标注"同构依据"

3. 联想过冷却（停规则）：
   从功能态推出第一层落处后 → 停
   给出2-3个并列候选（古代+现代混合），不继续往下推
   禁止在候选A上推"如果是A那可能还带X特征"

4. 查类象知识库验证一致性
```

**现代转译规则**（避免将功能态绑定在古代材质上）：

| 古类象 | 功能态（不变） | 古代落处 | 现代常见落处 |
|---|---|---|---|
| 鼎 | 容纳、调和 | 青铜鼎 | 瓶/罐/容器/锅具 |
| 革 | 去毛、变形 | 兽皮 | 注塑外壳/涂层/包覆物 |
| 苍筤竹 | 中空、管状 | 竹管 | 管线/塑料管/笔杆 |
| 离·大腹 | 中空有容 | 陶罐 | 电池仓/壳体/手机 |
| 巽·木/风 | 渗透、发散 | 草木 | 香料/挥发物/纤维 |
| 兑·金属 | 口、悦散 | 铜铁 | 接口/按键/塑料壳 |
| 坎·水 | 陷中流动 | 河流 | 液体/电流/电池液 |
| 震·雷 | 暴起、发声 | 雷声 | 电流/快充/震动 |
| 离·赤 | 红、亮 | 朱砂 | 指示灯/屏幕/红色涂层 |

### Layer 6：变复校验（B5）

```
变复验证（自校验）：
  · 爻变验证（如有动爻）：动爻阴阳翻转后变出何卦？同位置气态对比
  · 体复验证（默认）：上伏（外部变）+ 下伏（内部变）→ 二选一偏向
  · 参卦造化：互体合卦是什么？表里是否一致？

三原则：
  1. 气位非道德——"贵贱""小人"是气位描述，非道德褒贬
  2. 对待中见互补——每卦含其对岸
  3. 辞合不如义通——验证看气脉是否贯通，不看文字表面
```

## 3. 输出结构

```
📋 卦象总览
📊 卦画默观（上下卦德性 + 整体态势）
🔄 反对视角（反体/对体 + 形态判定）
🌊 卦脉流动（阴阳进退 + 消息定位 + 表里倚伏）
📅 月建参照（上下卦旺衰 + 体用力量对比）
🔗 落处展开（核心模式 + 多维翻译）
🔬 气脉校准（变复验证 + 校验结论）
💡 推演结论（三传时间线 + 启发提问）
```

**结束语（必须）**：
> 以上是根据卦象展开的可能性推演，并非确定性结论。请你结合自己的实际情况，判断哪一种可能性最符合你当下的处境。

## 4. 禁用语与鼓励语

### 严禁使用
- 断言词：一定、必然、毫无疑问、绝对
- 吉凶词：大吉、大凶、大利、大忌
- 指令词：宜、忌、应该、必须
- 承诺词：保证、包你、肯定能

### 鼓励使用
- 可能词：可能、或许、也许、有可能
- 象词：含有…之象、与…相呼应、呈现…态势
- 启发性提问："你目前实际感受到的是哪一种？"
