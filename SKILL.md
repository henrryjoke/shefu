## S1 调用与起卦

| 调用信号 | AI回应 | 启动信号 |
|---------|--------|---------|
| 「玩射覆」「射覆」「猜一猜」「帮我猜」 | 「好，我已准备好起卦。」 | 「开始」 |

## S2 两阶段交互

### Phase 1：特征画像（禁止品名）

```
v1 产出（仅本卦维度）：
  材质 🟡 / 形态 🟡 / 放置 🟡 / 开合 🟡
  ⛔ 禁止：品名、种类、候选列表、用途揣测

  玩家：「对了」→ 本卦维度 🟡→🟢 锁定
        「偏了」→ 纠偏
        「接近但XX不对」→ 聚焦维度

v2 产出（本卦确认后+互卦）：
  锁定: 材质 🟢 形态 🟢 放置 🟢 开合 🟢
  新增: 结构 🟡 / 内容物 🟡 / 隐藏特征 🟡

v3 产出（互卦确认后+变卦/动爻）：
  新增: 使用动态 🟡 / 终态 🟡 / 可动点 🟡
```

上限 5 轮，玩家说「够了」→直入 Phase 2。

### Phase 2：深化画像（提供品类+名称选项）

```
稳定画像 + 合法类象 + 月建系数
→ 品类：容器类/工具类/文具类/...
→ 名称：A(85%) / B(60%) / C(40%)
→ 命名权在玩家
```

### 卦脉初判（步骤0）

六子卦气脉定义（正易心法·陈抟注）：

| 卦 | 卦气 | 卦德 | 身体 |
|---|---|---|---|
| 震☳ | 阳在重重阴气之下涌动冲击 | 动 | 足 |
| 巽☴ | 阴滋长于阳之内部渗透滋长 | 入 | 手 |
| 坎☵ | 阳陷于阴质包夹中流行 | 陷 | 耳 |
| 离☲ | 阴附丽于阳中借阳四射 | 丽 | 目 |
| 艮☶ | 阳冒于阴之顶涌无可涌 | 止 | 鼻 |
| 兑☱ | 阴到达于外于表满溢四流 | 悦 | 口 |
| 乾☰ | 纯阳充盈健动不息 | 健 | — |
| 坤☷ | 纯阴承载空无静待 | 顺 | — |

0a-0f 流程见 shared/principles.md §3.0。

旺相休囚死以月建地支为准（见 shared/principles.md §3.0 旺相休囚死表）。

## S4 复盘与学习

每轮结束写入 `records/pending_updates/{日期}_{序号}.json`。

不自动更新正式数据。人工验证后合并：
```bash
python scripts/update_kb.py --review-pending
```

学习维度：月建校准 / 类象更新 / 四类规则强化 / 用户聚类。
