## 🧙 什么是射覆？

**射覆**（Shefu）——「射」者猜也，「覆」者盖也。一人藏物于覆器之下，另一人起卦推测覆下为何物。庄子曰：

> 射者非前期而中，谓之善射。  
> ——《庄子·田子方》

射覆见载于《汉书·东方朔传》。汉武帝尝以射覆试东方朔，朔起卦皆中，帝叹服。自此射覆成为汉代宫廷方士、文人之间流行的占卜游戏——**既是易学的功夫试炼，也是智力的雅趣对决。**

本技能继承五代宋初隐士**麻衣道者**《正易心法》及**陈抟**（希夷先生）注的卦脉解读传统，以「爻位即气位」为核心方法，**不查卦辞、不套口诀**，从六爻卦画的阴阳排列中直接读取气的态势，推演覆下之物的功能态与落处。

## 🎮 玩法说明

### 经典玩法（二人射覆）

```
玩家A：「起一卦，猜我手里握着什么。」
AI（射覆者）：起卦 → 默观卦画 → 推演物象 → 给出候选
玩家A：揭晓答案，验证卦脉
```

### 单人玩法（自测卦脉）

```
玩家：「起卦猜我左手边的杯子里有什么。」
AI：起卦并给出 2-3 个可能的物象候选
玩家：对照实际物品，看卦脉是否吻合
```

### 无限定玩法（开放式）

```
玩家：「起一卦，随便猜一个我家里的东西。」
AI：起卦后从卦脉结构推导功能态（健/顺/动/入/陷/丽/止/说），
     结合现代转译规则给出 2-3 个候选。
```

## 🧠 核心方法（三分钟入门）

### 原则一：爻位即气位

阳爻在哪里，阳气就在哪里；阴爻在哪里，阴气就在哪里。**不看标签，看位置。**

### 原则二：八卦是八种功能态

八卦不是名词，是德性（功能态）。从卦画结构直接读：

| ☰ 乾 | ☷ 坤 | ☳ 震 | ☴ 巽 | ☵ 坎 | ☲ 离 | ☶ 艮 | ☱ 兑 |
|---|---|---|---|---|---|---|---|
| 三阳 | 三阴 | 一阳**下** | 一阴**下** | 一阳**中** | 一阴**中** | 一阳**上** | 一阴**上** |
| **健** | **顺** | **动** | **入** | **陷** | **丽** | **止** | **悦** |

### 原则三：观不是推

先静观卦画结构——六爻阴阳怎么排列，气就怎么走。**观是直观照见，不是逻辑推导。**

### 原则四：同构容纳

同一气脉可落于多个维度——自然、人身、器物、人伦。**一脉多现。**

### 原则五：合于自然者为正

每个推演必须能从卦画结构回溯验证。方向不可逆：**结构 → 自然 → 语境**，不能从语境倒推。

## 🔮 调用与起卦

### 两种场景，两套信号

| 场景 | 调用信号（你说） | AI回应 | 你的启动信号 | AI执行 |
|------|----------------|--------|-------------|--------|
| **射覆（猜物）** | 「玩射覆」「射覆」「猜一猜」「帮我猜」 | 「好，我已准备好起卦。」 | 「开始」 | 大衍筮法起卦 |
| **问事（解卦）** | 「起卦」「测算」「算一算」「帮我算」「问卦」 | 「心里想的事清楚了吗？给几个关键词就好。」 | 任何回应（关键词/一句话/甚至「嗯」） | 大衍筮法起卦 |

**核心规则**：

- **调用信号** = 告诉 AI「我要做什么」（射覆还是问事）
- **启动信号** = 告诉 AI「可以开始了」（AI 才会执行起卦）
- **没有启动信号，AI 不起卦**——不会在你说"起卦"后就自作主张
- 你不说起卦方式 → 默认大衍筮法（不让你选、不让你摇）

### 起卦后

AI 静默执行大衍筮法（模拟蓍草49根，四营十八变），不展示计算过程。起卦完成后：

- **射覆** → 进入五阶段渐进式交互（方位定性→品类聚焦→形态锁定→颜色校准→揭晓复盘）
- **问事** → 进入七层分析链路（L1-L6）

### 如需特定起卦方式

| 你说 | AI做 |
|------|------|
| （不指定） | 大衍筮法（默认） |
| 「梅花易数起卦」 | 时间/数字起卦 |
| 「铜钱摇卦」 | 引导你逐爻手摇三枚硬币 |

## 📐 分析链路（七层）

```
L0 起卦 → A1 大衍筮法（默认）/ 梅花易数 / 六爻纳甲
L1 卦画默观 → 不看卦名，只看六爻阴阳
L2 反对分析 → 反体（主客翻转）/ 对体（阴阳全反）
L3 卦脉流动 → 阴阳进退 + 消息定位 + 互体表里
L4 月建加权 → 旺相休囚死（月令对卦气的强弱调节）
L5 落处展开 → 从功能态推导物象候选（2-3个）
L6 变复校验 → 爻变 + 体复 + 参卦（自校验）
```

## ⚠️ 使用须知

- **不判断吉凶**——不给出确定性结论，只展开可能性
- **禁用语**：一定、必然、大吉、大凶、宜、忌、应该
- **鼓励用语**：可能、或许、含有…之象、呈现…态势
- **结束语**——"以上是根据卦象展开的可能性推演，并非确定性结论。请你结合实际情况判断。"
