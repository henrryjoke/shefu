# 射覆 · 正易心法卦脉解读

> 「谛观」出品 · 基于麻衣道者《正易心法》42章·陈抟注

---

## 🧙 什么是射覆？

**射覆**（Shefu）——「射」者猜也，「覆」者盖也。一人藏物于覆器之下，另一人起卦推测覆下为何物。庄子曰：

> 射者非前期而中，谓之善射。  
> ——《庄子·田子方》

射覆见载于《汉书·东方朔传》。汉武帝尝以射覆试东方朔，朔起卦皆中，帝叹服。自此射覆成为汉代宫廷方士、文人之间流行的占卜游戏——**既是易学的功夫试炼，也是智力的雅趣对决。**

本技能继承五代宋初隐士**麻衣道者**《正易心法》及**陈抟**（希夷先生）注的卦脉解读传统，以「爻位即气位」为核心方法，**不查卦辞、不套口诀**，从六爻卦画的阴阳排列中直接读取气的态势，通过**约束求解 + 辨识问答**推演覆下之物。

## 🎮 玩法说明

### 经典玩法（二人射覆）

```
玩家A：「起一卦，猜我手里握着什么。」
AI（射覆者）：起卦 → 默观卦画 → 提取物性约束 → 辨识问答 → 给出候选
玩家A：揭晓答案，验证卦脉
```

### 单人玩法（自测卦脉）

```
玩家：「起卦猜我左手边的杯子里有什么。」
AI：起卦并进入三阶段约束求解
玩家：对照实际物品，看卦脉是否吻合
```

### 无限定玩法（开放式）

```
玩家：「起一卦，随便猜一个我家里的东西。」
AI：起卦后从卦脉结构提取物性向量，
    匹配候选集 → 辨识问答 → 给出 2-3 个候选。
```

### 提示机制（辅助聚焦）

如果你觉得 AI 的方向对但不精确，可以给一个**模糊提示**（不说具体物品）：

```
「提示：它和电力有关」  → AI 沿「坎=电」方向二次聚焦
「提示：跟我头有关」    → AI 搜索「乾=首，艮=鼻，兑=耳，离=目」
「提示：它比较小」      → AI 排除大件物品类
```

## 🧠 核心方法（三分钟入门）

### 原则一：爻位即气位

阳爻在哪里，阳气就在哪里；阴爻在哪里，阴气就在哪里。**不看标签，看位置。**

### 原则二：八卦是八种功能态

八卦不是名词，是德性（功能态）。从卦画结构直接读：

| ☰ 乾 | ☷ 坤 | ☳ 震 | ☴ 巽 | ☵ 坎 | ☲ 离 | ☶ 艮 | ☱ 兑 |
|---|---|---|---|---|---|---|---|
| 三阳 | 三阴 | 一阳**下** | 一阴**下** | 一阳**中** | 一阴**中** | 一阳**上** | 一阴**上** |
| **健** | **顺** | **动** | **入** | **陷** | **丽** | **止** | **悦** |

### 原则三：观不是推

先静观卦画结构——六爻阴阳怎么排列，气就怎么走。**观是直观照见，不是逻辑推导。**

### 原则四：同构容纳

同一气脉可落于多个维度——自然、人身、器物、人伦。**一脉多现。**

### 原则五：合于自然者为正

每个推演必须能从卦画结构回溯验证。方向不可逆：**结构 → 自然 → 语境**，不能从语境倒推。

## 🔮 调用与起卦

### 两种场景，两套信号

| 场景 | 调用信号（你说） | AI回应 | 你的启动信号 | AI执行 |
|------|----------------|--------|-------------|--------|
| **射覆（猜物）** | 「玩射覆」「射覆」「猜一猜」「帮我猜」 | 「好，我已准备好起卦。」 | 「开始」 | 大衍筮法起卦 |
| **问事（解卦）** | 「起卦」「测算」「算一算」「帮我算」「问卦」 | 「心里想的事清楚了吗？给几个关键词就好。」 | 任何回应（关键词/一句话/甚至「嗯」） | 大衍筮法起卦 |

**核心规则**：

- **调用信号** = 告诉 AI「我要做什么」（射覆还是问事）
- **启动信号** = 告诉 AI「可以开始了」（AI 才会执行起卦）
- **没有启动信号，AI 不起卦**——不会在你说"起卦"后就自作主张
- 你不说起卦方式 → 默认大衍筮法（不让你选、不让你摇）

### 三阶段交互流程（射覆专用）

```
S1 — 第一印象（卦脉直读）
  AI 给出：
    · 卦象总览（本卦/变卦/动爻）
    · 功能态直读（卦的结构在说什么）
    · 物性约束向量（材质/形态/功能/结构/颜色/尺寸倾向）
    · 置信度分级（🟢高 / 🟡中 / 🔴低）
  
  你可以：
    → 直接揭晓答案（如果AI猜对了方向）
    → 说「继续」或「猜吧」（进入S2）
    → 给一个模糊提示（帮助AI聚焦方向）

S2 — 辨识问答（约束求解）
  AI 问 1-2 个辨识问题（卦象无法给出的维度）
    问题维度：整体大小 / 重量感 / 厚度 / 表面纹理 / 用途 / 是否有附属件
  
  你可以：
    → 如实回答（AI据此排除 → 最终候选）
    → 说「直接猜」（跳过问答，给出候选）
    → 揭晓答案（任何时候都可以）

S3 — 揭晓与复盘
  AI 回溯验证：哪些信号命中了？哪些遗漏了？为什么？
  更新映射表
```

### 如需特定起卦方式

| 你说 | AI做 |
|------|------|
| （不指定） | 大衍筮法（默认） |
| 「梅花易数起卦」 | 时间/数字起卦 |
| 「铜钱摇卦」 | 引导你逐爻手摇三枚硬币 |

## 📐 分析链路（四层递进约束求解）

```
L0 起卦 → 大衍筮法（默认）/ 梅花易数 / 六爻纳甲

L1 卦画默观
  不看卦名，只看六爻阴阳排列。
  · 上卦下卦各是什么功能态（健/顺/动/入/陷/丽/止/说）
  · 阳爻在哪里，阴爻在哪里——气就怎么走
  · 重卦判断：上下卦相同 → 双体/对称结构信号

L2 卦脉解构
  从功能态 + 结构信号 → 提取物性约束向量。
  · 反对分析：本卦→变卦揭示物品的核心动态特征
  · 卦脉流动：阴阳进退、消息定位、互体表里
  · 月建加权：旺相休囚死 → 材质/颜色时令调节
  · 动爻定位：动爻位置 → 变化部位（见注1）
  · 颜色系统：月建→外壳/手柄色；日建→功能部/内部色
  
  → 输出物性约束向量：
    [材质, 形态, 功能, 结构, 颜色, 尺寸倾向]

L3 约束求解 ⭐（核心创新）
  3.1 类空间生成
      用物性向量扫描十大品类（见《射覆映射手册》§5.1），
      对每个候选计算约束满足度（每个维度 ✓/✗）。
      不分古代/现代，平等列入。
  
  3.2 置信度分级
      🟢 高置信：2个以内高匹配候选（>80%满足度）
         → 直接给出候选，不进入问答
      🟡 中置信：3-5个候选
         → 进入辨识问答（见3.3）
      🔴 低置信：6+个候选或匹配<50%
         → 建议用户给提示，或换物重占
  
  3.3 辨识问答（仅🟡中置信触发）
      对剩余候选问1-2个卦象无法给出的辨识维度：
        大小：尺寸/体积
        重量：轻/重/有分量感
        厚度：薄片状/块状/立体
        表面：光滑/粗糙/有纹理/有凹凸
        用途：装饰品/工具/容器/食品
        附属件：有绳子/有盖子/有按键
      
      规则：
        · 问题二选一或对比性（帮AI聚焦，不是让你盲猜）
        · 最多2个问题后必须给出最终候选
        · 用户任何时候可以说「直接猜」跳过问答
  
  3.4 加权判定
      根据回答排除不符合的候选 → 给出2-3个加权候选

L4 变复校验
  爻变 + 体复 + 参卦（自校验）。
  · 验证推导一致性，排除矛盾
  · 生成复盘用的回溯路径

注1：动爻位置与物理对应
  初爻=底部/根基  二爻=下部  三爻=中下部
  四爻=中上部      五爻=上部  上爻=顶部/尖端/最外层
  阳→阴动=硬→软/实→虚/闭→开
  阴→阳动=软→硬/虚→实/开→闭
  ⚠️ 特别注意：动爻可能指示机械运动（铰链/折叠/开合/旋转/滑动）
```

## 🎯 输出示例（约束求解模式）

```
📋 卦象总览
  本卦：☶☶ 艮为山 · 变卦：☷☶ 地山谦
  动爻：上六阳→阴动

📊 卦画默观（L1）
  上下皆艮☶（止）—— 重卦结构 → 双体对称
  一阳在上，整体态势：两个硬质、静止的对称体

🔬 卦脉解构（L2）
  反对：艮→谦，上六阳→阴动
    · 上爻=顶端/最外层，阳→阴=硬→软、闭→开
    · → 顶端有开合机制（铰链运动）
  
  互体：雷水解 ☳☵
    · 震=动（内部有启动力）+ 坎=陷（中空/锋利）
  
  月建：巳火 → 外壳/手柄 → 偏暖色（红/橙）
  日建：申金 → 功能部/内部 → 偏银白

  物性约束向量：
    材质：硬质（艮=土石），可能含金属（坎在互体=险/锋利）
    形态：双体对称 / 框状（艮=门阙）/ 环形手柄
    功能：常态静止 / 顶端可开合（上爻动）
    颜色：手柄暖色 + 功能部银白
    尺寸：手持工具级

🔍 约束求解（L3）
  匹配向量 [硬质+双体+框状+环形+开合+手持] → 扫描十大品类：

     剪刀 ✂️    [硬✓ 双✓ 框✓ 环✓ 开合✓] → 5/5
     钳子 🔧    [硬✓ 双✓ 框✓ 环✓ 开合✓] → 5/5
     镊子       [硬✓ 双✓ 框✓ 环✓ 开合✓] → 5/5
     眼镜 👓    [硬✓ 双✓ 框✓ 环✗ 开合✓] → 4/5

  🟡 中置信（3个高匹配候选），需要辨识：

  → 「它有两个对称的操作柄，工作端是刀刃还是钳口？」

  （等你回答后，排除不符合的 → 给出最终候选）

✅ 变复校验（L4）
  艮→谦 = 止极而退藏 → 物品常态静止，使用时有变化
  上爻动 = 变化在顶端 → 开合/铰链运动 ✅ 自洽
```

## ⚠️ 使用须知

- **不判断吉凶**——不给出确定性结论，只展开可能性
- **不盲目猜物**——卦象给约束，辨识问答补足缺口
- **禁用语**：一定、必然、大吉、大凶、宜、忌、应该
- **鼓励用语**：可能、或许、含有…之象、呈现…态势
- **结束语**——"以上是根据卦象展开的可能性推演，并非确定性结论。请你结合实际情况判断。"
