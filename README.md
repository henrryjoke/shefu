# 射覆 — 正易心法卦脉解读 Skill

「谛观」出品。基于麻衣道者《正易心法》42 章 · 陈抟注的卦脉解读方法。

## 核心原理

- **爻位即气位** — 阳爻在哪里，阳气就在哪里
- **八卦是德性** — 健顺动入陷丽止说，从结构中直接读出
- **观不是推** — 先静观卦画，后做语境翻译
- **同构容纳** — 一气脉多落处，自然/人伦/治理一脉多现
- **合于自然者为正** — 每个判断可从卦画结构回溯验证

## 起卦方式（v0.2.1）

| 方法 | 起卦方式 | 适用 |
|------|---------|------|
| 大衍筮法（默认） | 模拟蓍草49根，四营十八变 | 自动起卦，Agent 静默执行 |
| 梅花易数 | 当前时间起卦 / 数字起卦 | 用户明确指定「梅花」时使用 |
| 六爻纳甲 | 用户手摇三枚铜钱，Agent 逐爻引导 | 用户明确指定「铜钱」「摇卦」时使用 |

## 七层分析链路

| 层 | 名称 | 功能 |
|---|---|---|
| L0 | 起卦引擎 | 大衍筮法（默认）/ 梅花易数（时间/数字）/ 六爻纳甲（铜钱+纳甲装爻）|
| L1 | 卦画默观 | 六爻阴阳→上下卦德性→整体态势 |
| L2 | 反对分析 | 反体(主客)+对体(对待)+四维合观 |
| L3 | 卦脉流动 | 阴阳进退+消息定位+中爻造化 |
| L4 | 月建加权 | 旺相休囚死→调节气的强弱 |
| L5 | 落处展开 | 同构容纳+多维翻译+现代转译 |
| L6 | 变复校验 | 变复验证+参卦造化+自校验 |

## 使用方式

```
起卦：起一卦，问项目决策     ← 默认大衍筮法
梅花：梅花易数起卦，问感情   ← 明确指定梅花时
六爻：帮我六爻摇卦，问健康   ← 明确指定铜钱摇卦时
解卦：正易心法分析山水蒙卦，职业变更
```

## 不判

不判断吉凶，不给出确定性结论。展开可能性，启发用户自主判断。
