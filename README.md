# 射覆 - 正易心法卦脉解读

> "射覆"（shè fù）是中国古代一种雅致又烧脑的猜物游戏——把一件东西藏在瓯、盒之类的器皿下面，让人通过占卜、推理或联想来猜里面究竟是什么。
>
> "射"是猜度、推断，"覆"是覆盖、遮盖。在古代，射覆与《易经》、五行八卦紧密挂钩。《汉书·东方朔传》记载，汉武帝让人把壁虎藏在盂下让东方朔猜，东方朔通过占卜精准猜中。在文人雅士的行酒令中，这是展示学识和卜筮能力的高级娱乐。
>
> 放到现代，你可以把它理解为**古代版的"开盲盒"**，只不过开盒之前需要动用易学知识或逻辑联想能力来"算"出答案。在术数爱好者圈子里，射覆至今仍是训练卦象直觉和逻辑推理的趣味练习。

基于麻衣道者《正易心法》42 章 · 陈抟（希夷）注的卦脉解卦方法。
从卦画结构直接读取气的态势，不查表、不套口诀。

## 核心原理

- **爻位即气位** — 阳爻在哪里，阳气就在哪里
- **八卦是德性** — 健顺动入陷丽止说，从结构中直接读出
- **观不是推** — 先静观卦画，后做语境翻译
- **同构容纳** — 一气脉多落处，自然/人伦/治理一脉多现
- **合于自然者为正** — 每个判断可从卦画结构回溯验证

| 层 | 名称 | 功能 |
|---|---|---|
| L1 | 卦画默观 | 六爻阴阳→上下卦德性→整体态势 |
| L2 | 反对分析 | 反体(主客)+对体(对待)+四维合观 |
| L3 | 卦脉流动 | 阴阳进退+消息定位+中爻造化 |
| L4 | 月建加权 | 旺相休囚死→调节气的强弱 |
| L5 | 落处展开 | 同构容纳+多维翻译+现代转译 |
| L6 | 变复校验 | 变复验证+参卦造化+自校验 |
